详细描述

Use this as a condition-blocking yield instruction for Unity Coroutines. The routine will pause until the AnimationState.TrackEntry fires its Complete event. It can be configured to trigger on the End event as well to cover interruption.

See the Spine Unity Events documentation page and http://esotericsoftware.com/spine-api-reference#AnimationStateListener for more information on when track events will be triggered.
