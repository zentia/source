static void XSUM_benchMem ( const void * buffer,
size_t bufferSize )
static

XSUM_benchMem(): Benchmark provided content up to twice per function:

  • once at provided aligned memory address (%16)
  • second time at unaligned memory address (+3) Enabled functions and modes are provided via @g_hashesToBench global variable. @buffer: Must be 16-byte aligned. The allocated size of underlying @buffer must be >= (@bufferSize+3). This function also fills @g_benchSecretBuf, to bench XXH3's _withSecret() variants.
