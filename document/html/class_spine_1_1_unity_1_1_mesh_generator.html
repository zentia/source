inlinestaticpackage

Step 2 of solving tangents. Fills (part of) a temporary tangent-solution buffer based on the vertices and uvs defined by a submesh's triangle buffer. Only needs to be called once for single-submesh meshes.

参数
tempTanBufferA temporary Vector3[] for calculating tangents.
verticesThe mesh's current vertex position buffer.
trianglesThe mesh's current triangles buffer.
uvsThe mesh's current uvs buffer.
vertexCountNumber of vertices that require tangents (or the size of the vertex array)
triangleCountThe number of triangle indexes in the triangle array to be used.
