详细描述

template<typename T>
class tf::UnboundedTaskQueue< T >

class to create a lock-free unbounded work-stealing queue

模板参数
Tdata type (must be a pointer type)

This class implements the work-stealing queue described in the paper, Correct and Efficient Work-Stealing for Weak Memory Models.

Only the queue owner can perform pop and push operations, while others can steal data from the queue simultaneously.

◆ steal_with_hint()

template<typename T>
T tf::UnboundedTaskQueue< T >::steal_with_hint ( size_t & num_empty_steals)

attempts to steal a task with a hint mechanism

参数
num_empty_stealsa reference to a counter tracking consecutive empty steal attempts

This function tries to steal a task from the queue. If the steal attempt is successful, the stolen task is returned. Additionally, if the queue is empty, the provided counter num_empty_steals is incremented; otherwise, num_empty_steals is reset to zero.
