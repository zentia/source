Release Notes

This page summarizes the release notes of Taskflow. We classify each release with three numbers:

Major.Minor.Patch

A major release indicates significant codebase changes and API modifications, a minor release indicates technical improvement over a major release line, and a patch release indicates fixes of bugs and other issues.

All releases are available in @ProjectGitHub.
