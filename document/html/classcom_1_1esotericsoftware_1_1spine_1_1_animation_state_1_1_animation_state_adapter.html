void com.esotericsoftware.spine.AnimationState.AnimationStateAdapter.complete ( TrackEntry entry)
inline

Invoked every time this entry's animation completes a loop. This may occur during mixing (after interrupt(TrackEntry)).

If this entry's TrackEntry#getMixingTo() is not null, this entry is mixing out (it is not the current entry).

Because this event is triggered at the end of AnimationState#apply(Skeleton), any animations set in response to the event won't be applied until the next time the AnimationState is applied.

Invoked when this entry will be disposed. This may occur without the entry ever being set as the current entry.

References to the entry should not be kept after dispose is called, as it may be destroyed or reused.
