详细描述

Atlas data loaded from a .atlas file and its corresponding .png files. For each atlas image, a corresponding Bitmap and Paint is constructed, which are used when rendering a skeleton that uses this atlas.

Use the static methods AndroidTextureAtlas#fromAsset(String, Context), AndroidTextureAtlas#fromFile(File), and AndroidTextureAtlas#fromHttp(URL, File) to load an atlas.
