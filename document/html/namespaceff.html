◆ combine_nodes_in_pipeline()

static const ff_pipeline ff::combine_nodes_in_pipeline ( ff_node & node1,
ff_node & node2,
bool cleanup1 = false,
bool cleanup2 = false )
inlinestatic

combines two stages returning a pipeline:

  • node1 and node2 standard nodes (or ff_comb) --> pipeline(node1, node2)
  • node1 standard node and node2 is a farm --> pipeline(node2) (node1 is merged with node2's emitter)
  • node1 is a farm and node2 is a standard node --> pipeline(node1) (node2 is merged with node1's collector)
  • node1 and node2 are both farms --> pipeline(node1, node2) (collector is merged with emitter – see case4.2 of combine_farms) (NOTE: if node1 is an ordered farm, then its collector is not removed)
