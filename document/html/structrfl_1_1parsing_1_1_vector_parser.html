template<class R, class W, class VecType, class ProcessorsType>
requires AreReaderAndWriter<R, W, VecType>
struct rfl::parsing::VectorParser< R, W, VecType, ProcessorsType >

This can be used for data structures that would be expressed as array in serialized format (std::vector, std::set, std::deque, ...), but also includes map-like types, when the key is not of type std::string.
