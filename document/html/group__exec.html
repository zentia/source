◆ tpc_device_job_launch()

tpc_res_t tpc_device_job_launch ( tpc_dev_ctx_t * dev_ctx,
tpc_job_id_t const job_id,
tpc_device_job_launch_flag_t const flags )

Launches the given job and releases its id (does not affect alloc'ed handles, means only that kernel arguments can no longer be set using this id). Blocks caller execution until kernel has finished.

参数
dev_ctxdevice context
job_idjob id
flagslaunch flags, e.g., TPC_DEVICE_JOB_LAUNCH_BLOCKING
返回
TPC_SUCCESS if execution was successful and results can be retrieved, TPC_FAILURE otherwise.
