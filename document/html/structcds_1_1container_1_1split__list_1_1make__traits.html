template<typename... Options>
struct cds::container::split_list::make_traits< Options >

Metafunction converting option list to traits struct

Available Options:

  • split_list::ordered_list - a tag for ordered list implementation.
  • split_list::ordered_list_traits - type traits for ordered list implementation. For MichaelList use container::michael_list::traits or derivatives, for LazyList use container::lazy_list::traits or derivatives.
  • plus any option from intrusive::split_list::make_traits
