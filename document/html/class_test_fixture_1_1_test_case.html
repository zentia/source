详细描述

template<typename ConcreteTestFixture>
class TestFixture< ConcreteTestFixture >::TestCase

Wrapper for the test methods of concrete TestFixtures.

Makes the 'Leave' role in the 'Composite' GoF pattern because can't be be a composition of other tests.

It's also a case of 'Command' pattern because it encapsules in an object certain functionality whose execution depends on some deferred entity.
