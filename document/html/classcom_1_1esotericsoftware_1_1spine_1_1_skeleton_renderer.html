◆ draw() [1/3]

void com.esotericsoftware.spine.SkeletonRenderer.draw ( Batch batch,
Skeleton skeleton )
inline

Renders the specified skeleton. If the batch is a PolygonSpriteBatch, draw(PolygonSpriteBatch, Skeleton) is called. If the batch is a TwoColorPolygonBatch, draw(TwoColorPolygonBatch, Skeleton) is called. Otherwise the skeleton is rendered without two color tinting and any mesh attachments will throw an exception.

This method may change the batch's blending function. The previous blend function is not restored, since that could result in unnecessary flushes, depending on what is rendered next.

If true, blend modes for premultiplied alpha will be used. Set to false if premultiplied alpha is not being used. Default is false.

◆ setPremultipliedAlphaColors()

void com.esotericsoftware.spine.SkeletonRenderer.setPremultipliedAlphaColors ( boolean pmaColors)
inline

If true, colors will be multiplied by their alpha before being sent to the GPU. Set to false if premultiplied alpha is not being used or if the shader does the multiplication (libgdx's default batch shaders do not). Default is false.
