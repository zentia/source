template<typename... Options>
struct cds::intrusive::fcqueue::make_traits< Options >

Metafunction converting option list to traits

Options are:

  • any cds::algo::flat_combining::make_traits options
  • opt::disposer - the functor used to dispose removed items. Default is opt::intrusive::v::empty_disposer. This option is used only in FCQueue::clear() function.
  • opt::stat - internal statistics, possible type: fcqueue::stat, fcqueue::empty_stat (the default)
  • opt::enable_elimination - enable/disable operation elimination By default, the elimination is disabled (false)
