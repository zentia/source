6// This file implements a priority_queue that is just like the C++
7// std::priority_queue adapter class, except it has a couple extension functions.
8// The primary distinctions between this priority_queue and std::priority_queue are:
9// - priority_queue has a couple extension functions that allow you to
10// use a priority queue in extra ways. See the code for documentation.
11// - priority_queue can contain objects with alignment requirements.
12// std::priority_queue cannot do so without a bit of tedious non-portable effort.
13// - priority_queue supports debug memory naming natively.
14// - priority_queue is easier to read, debug, and visualize.
15// - priority_queue is savvy to an environment that doesn't have exception handling,
16// as is sometimes the case with console or embedded environments.
17//
