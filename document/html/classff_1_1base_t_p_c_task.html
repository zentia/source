template<typename TaskT_in, typename TaskT_out = TaskT_in>
virtual TaskT_out * ff::baseTPCTask< TaskT_in, TaskT_out >::releaseTask ( TaskT_in * t)
inlinevirtual

Releases the input task just computed. This method is called at the of the task computation. It may be used to perform per-task host memory cleanup (i.e. releasing the host memory previously allocated in the setTask function) or to execute a post-elaboration phase

参数
tinput task
返回
the output task

Sets the host-pointer to the input parameter. The order of calls determin the order of parametes.

参数
inPtrthe host-pointer
sizethe number of elements in the input array (bytesize=size*sizeof(ptrT))
copyif CopyFlags::COPY is set the data will be copied into the device
reuseif BitFlags::REUSE is set then the run-time looks for a previously allocated device handle associated with the inPtr host pointer
releaseif BitFlags::RELEASE is set the device memory will be released at the end of the kernel execution
