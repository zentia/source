◆ computeWorldVertices()

void com.esotericsoftware.spine.attachments.RegionAttachment.computeWorldVertices ( Slot slot,
float[] worldVertices,
int offset,
int stride )
inline

Transforms the attachment's four vertices to world coordinates. If the attachment has a sequence, the region may be changed.

See World transforms in the Spine Runtimes Guide.

参数
worldVerticesThe output world vertices. Must have a length >= offset + 8.
offsetThe worldVertices index to begin writing values.
strideThe number of worldVertices entries between the value pairs written.
