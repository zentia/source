◆ try_increase_load()

int rml::internal::omp_connection_v2::try_increase_load ( size_type ,
bool  )
privatevirtual

Request n coins. Returns number of coins granted. Oversubscription amount if negative.

Always granted if is_strict is true.

  • Positive or zero result indicates that the number of coins was taken from the bank.
  • Negative result indicates that no coins were taken, and that the bank has deficit by that amount and the caller (if being a good citizen) should return that many coins.
