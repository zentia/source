详细描述

template<unsigned NT, unsigned VT>
class tf::syclExecutionPolicy< NT, VT >

class to define execution policy for SYCL standard algorithms

模板参数
NTnumber of threads per block
VTnumber of work units per thread

Execution policy configures the kernel execution parameters in SYCL algorithms. The first template argument, NT, the number of threads per block should always be a power-of-two number. The second template argument, VT, the number of work units per thread is recommended to be an odd number to avoid bank conflict.

Details can be referred to SYCLSTDExecutionPolicy.
