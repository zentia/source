◆ RunTestImpl()

void Perf::internal::RunTestImpl ( TestResults & tr,
RunConfig & cfg,
RunMemFnPtr pfnTest,
TimingSeries & ts )

Runs the test function, does statistical processing, and, if title is nonzero, prints results.

If histogramFileName is a string, the histogram of individual runs is generated and stored in a file with the given name. If it is NULL then the histogram is printed on the console. By default no histogram is generated. The histogram format is: "rate bucket start" "number of tests in this bucket".
