Metafunction converting option list to traits struct

Available Options:

  • opt::hash - mandatory option, specifies hash functor.
  • opt::item_counter - optional, specifies item counting policy. See traits::item_counter for default type.
  • opt::allocator - optional, bucket table allocator. Default is CDS_DEFAULT_ALLOCATOR.
