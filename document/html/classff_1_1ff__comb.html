void ff::ff_comb::eosnotify ( ssize_t = -1)
inlineprotectedvirtual

EOS callback

This method is called when an EOS has just been received from one input channel. Inside this method it is possible to call ff_send_out to produce data elements in output (this is not possible in the svc_end method). The parameter

参数
idis the ID of the channel that received the EOS.
