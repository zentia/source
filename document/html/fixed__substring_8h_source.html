217 // Partially supported functionality
218 //
219 // When using fixed_substring on a character sequence that is within another
220 // string, the following functions may do one of two things:
221 // 1 Attempt to reallocate
222 // 2 Write a 0 char at the end of the fixed_substring
223 //
224 // Item #1 will result in a crash, due to the attempt by the underlying
225 // string class to free the substring memory. Item #2 will result in a 0
226 // char being written to the character array. Item #2 may or may not be
227 // a problem, depending on how you use fixed_substring. Thus the following
228 // functions cannot be used safely.
229
