详细描述

template<class T, Copyability C = Copyability::NON_COPYABLE>
class rfl::Box< T, C >

The Box class behaves very similarly to the unique_ptr, but unlike the unique_ptr, it is 100% guaranteed to be filled at all times (unless the user tries to access it after calling std::move does something else that is clearly bad practice).

By default Box behaves like a unique_ptr in relation to copying, but it can be configured to add copy constructor and assignment operators that call the same function of the contained type T.
