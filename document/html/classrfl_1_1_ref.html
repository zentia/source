详细描述

template<class T>
class rfl::Ref< T >

The Ref class behaves very similarly to the shared_ptr, but unlike the unique_ptr, it is 100% guaranteed to be filled at all times (unless the user tries to access it after calling std::move does something else that is clearly bad practice).
