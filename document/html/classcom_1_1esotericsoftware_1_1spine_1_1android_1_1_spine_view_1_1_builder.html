Instantiate a Builder used to build a SpineView, which is a View to display a Spine skeleton.

参数
controllerThe skeleton displayed by a SpineView can be controlled via a SpineController.

SpineView com.esotericsoftware.spine.android.SpineView.Builder.build ( )
inline

Builds a new SpineView.

After initialization is complete, the provided SpineController is invoked as per the SpineController semantics, to allow modifying how the skeleton inside the widget is animated and rendered.
