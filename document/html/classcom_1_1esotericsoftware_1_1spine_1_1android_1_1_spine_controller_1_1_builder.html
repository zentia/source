◆ Builder()

com.esotericsoftware.spine.android.SpineController.Builder.Builder ( SpineControllerCallback onInitialized)
inline

Instantiate a Builder used to build a SpineController, which controls how the skeleton of a SpineView is animated and rendered. Upon initialization of a SpineView, the provided onInitialized callback method is called once. This method can be used to set up the initial animation(s) of the skeleton, among other things.

参数
onInitializedUpon initialization of a SpineView, the provided onInitialized callback method is called once. This method can be used to set up the initial animation(s) of the skeleton, among other things.

Builder com.esotericsoftware.spine.android.SpineController.Builder.setOnAfterUpdateWorldTransforms ( SpineControllerCallback onAfterUpdateWorldTransforms)
inline

Sets the onAfterUpdateWorldTransforms callback. This method is called after the skeleton's current pose is calculated using Skeleton#updateWorldTransform(Skeleton.Physics). It can be used to modify the current pose before rendering the skeleton.

Sets the onBeforePaint callback. It is called before the skeleton's current pose is rendered by the SpineView. It allows rendering backgrounds or other objects that should go behind the skeleton on the Canvas.
