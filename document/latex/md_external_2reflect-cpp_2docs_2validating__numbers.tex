reflect-\/cpp can also be used to impose condition on numbers. For instance, if you want an integer that is greater or equal to 10, you can do the following\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{\textcolor{keyword}{using\ }IntGreaterThan10\ =\ \mbox{\hyperlink{structrfl_1_1_validator}{rfl::Validator<int,\ rfl::Minimum<10>}}>;}

\end{DoxyCode}


When you then use the type {\ttfamily Int\+Greater\+Than10} inside you {\ttfamily \doxylink{structrfl_1_1_field}{rfl\+::\+Field}}, the condition will be automatically validated.

The underlying value can be retrieved using the {\ttfamily .\doxylink{imgui__impl__opengl3__loader_8h_abe8e049f756b5ba547bda825af81b645}{value()}} method.
