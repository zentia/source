Get current processor number 

In HP-\/\+UX, processor and locality domain IDs are not guaranteed to exist in numerical order. There may be holes in a sequential list of IDs.

This function guarantees sequence of processor numbers using internal processor ID mapping. It returns sequential processor number, ie a number from 0 to N -\/ 1, where N -\/ processor count. 函数调用图\+:
