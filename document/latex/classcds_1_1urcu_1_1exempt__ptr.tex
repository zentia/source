class cds\+::urcu\+::exempt\+\_\+ptr$<$ RCU, Node\+Type, Value\+Type, Disposer, Cast $>$}
Exempt pointer for RCU 

This special pointer class is intended for returning extracted node from RCU-\/based container. The destructor (and {\ttfamily \doxylink{classcds_1_1urcu_1_1exempt__ptr_ab21b8b42a14821ae94293c328017420c}{release()}} member function) invokes {\ttfamily RCU\+::retire\+\_\+ptr$<$ Disposer $>$()} function to dispose the node. For non-\/intrusive containers from {\ttfamily \doxylink{namespacecds_1_1container}{cds\+::container}} namespace {\ttfamily Disposer} is usually an invocation of node deallocator. For intrusive containers the disposer can be empty or it can trigger an event "{}node can be reused safely"{}. In any case, the exempt pointer concept keeps RCU semantics. The destructor or {\ttfamily \doxylink{classcds_1_1urcu_1_1exempt__ptr_ab21b8b42a14821ae94293c328017420c}{release()}} should be called outside the RCU lock of current thread.

You don\textquotesingle{}t need use this helper class directly. Any RCU-\/based container typedefs a simplified version of this template.

Template arguments\+:
\begin{DoxyItemize}
\item {\ttfamily RCU} -\/ one of \doxylink{namespacecds_1_1urcu_cds_urcu_gc}{RCU type}
\item {\ttfamily Node\+Type} -\/ container\textquotesingle{}s node type
\item {\ttfamily \doxylink{test__parallel__reduce_8cpp_aac3e1a5e422a80a86b55a5c15b106ba9}{Value\+Type}} -\/ value type stored in container\textquotesingle{}s node. For intrusive containers it is the same as {\ttfamily Node\+Type} 
\item {\ttfamily Disposer} -\/ a disposer functor
\item {\ttfamily Cast} -\/ a functor for casting from {\ttfamily Node\+Type} to {\ttfamily \doxylink{test__parallel__reduce_8cpp_aac3e1a5e422a80a86b55a5c15b106ba9}{Value\+Type}}. For intrusive containers the casting is usually disabled, i.\+e. {\ttfamily Cast} is {\ttfamily void}. 
\end{DoxyItemize}
