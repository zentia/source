\begin{DoxyParagraph}{About item disposing}
The Michael \& Scott\textquotesingle{}s queue algo has a key feature\+: even if the queue is empty it contains one item that is "{}dummy"{} one from the standpoint of the algo. See {\ttfamily \doxylink{classcds_1_1intrusive_1_1_m_s_queue_acbfac1d0165a1060d4d4f82dea4b44ec}{dequeue()}} function for explanation.
