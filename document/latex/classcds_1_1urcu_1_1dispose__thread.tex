Start reclamation cycle 

This function is called by \doxylink{classcds_1_1urcu_1_1general__threaded}{general\+\_\+threaded} object to notify the reclamation thread about a new work. {\ttfamily buf} buffer contains retired objects ready to free. The reclamation thread should free all {\ttfamily buf} objects {\ttfamily m\+\_\+n\+Epoch} field of which is no more than {\ttfamily n\+Cur\+Epoch}.
