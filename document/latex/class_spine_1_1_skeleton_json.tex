\doxysubsection{详细描述}
Loads skeleton data in the \doxylink{namespace_spine}{Spine} JSON format. 

JSON is human readable but the binary format is much smaller on disk and faster to load. See \doxylink{class_spine_1_1_skeleton_binary}{Skeleton\+Binary}.

See \href{http://esotericsoftware.com/spine-json-format}{\texttt{ Spine JSON format}} and \href{http://esotericsoftware.com/spine-loading-skeleton-data\#JSON-and-binary-data}{\texttt{ JSON and binary data}} in the \doxylink{namespace_spine}{Spine} Runtimes Guide.
