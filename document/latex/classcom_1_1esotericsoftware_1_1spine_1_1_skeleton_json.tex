\doxysubsection{详细描述}
Loads skeleton data in the \doxylink{namespace_spine}{Spine} JSON format. 

JSON is human readable but the binary format is much smaller on disk and faster to load. See {\ttfamily \doxylink{classcom_1_1esotericsoftware_1_1spine_1_1_skeleton_binary}{Skeleton\+Binary}}. 

See \href{https://esotericsoftware.com/spine-json-format}{\texttt{ Spine JSON format}} and \href{https://esotericsoftware.com/spine-loading-skeleton-data\#JSON-and-binary-data}{\texttt{ JSON and binary data}} in the \doxylink{namespace_spine}{Spine} Runtimes Guide.
