\doxysubsection{详细描述}
class to create an immutable view of a worker 

An executor keeps a set of internal worker threads to run tasks. \doxylink{bench__gemm_8cpp_addc86e8508f14411ec98f521c520f875}{A} worker view provides users an immutable interface to observe when a worker runs a task, and the view object is only accessible from an observer derived from \doxylink{classtf_1_1_observer_interface}{tf\+::\+Observer\+Interface}.
