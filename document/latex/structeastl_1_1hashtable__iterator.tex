\doxylink{structeastl_1_1hashtable__iterator}{hashtable\+\_\+iterator}

A \doxylink{structeastl_1_1hashtable__iterator}{hashtable\+\_\+iterator} iterates the entire hash table and not just nodes within a single bucket. Users in general will use a hash table iterator much more often, as it is much like other container iterators (e.\+g. \doxylink{classeastl_1_1vector_a4bae2bde2429116a3cb0e31b2168ccbe}{vector\+::iterator}).

The b\+Const parameter defines if the iterator is a const\+\_\+iterator or an iterator.
