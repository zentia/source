\subsubsection*{template$<$class \mbox{\hyperlink{_tutorial__sparse__example_8cpp_af639fb96631ba0018f0b2259fc95edfb}{T}}, \mbox{\hyperlink{namespacerfl_a5963cbba771224f05b872949a5cc175c}{Copyability}} C = Copyability\+::\+NON\+\_\+\+COPYABLE$>$\newline
class rfl\+::\+Box$<$ T, C $>$}
The \doxylink{classrfl_1_1_box}{Box} class behaves very similarly to the unique\+\_\+ptr, but unlike the unique\+\_\+ptr, it is 100\% guaranteed to be filled at all times (unless the user tries to access it after calling std\+::move does something else that is clearly bad practice).

By default \doxylink{classrfl_1_1_box}{Box} behaves like a unique\+\_\+ptr in relation to copying, but it can be configured to add copy constructor and assignment operators that call the same function of the contained type T.
