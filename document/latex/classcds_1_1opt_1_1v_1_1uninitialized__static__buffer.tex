class cds\+::opt\+::v\+::uninitialized\+\_\+static\+\_\+buffer$<$ T, Capacity, Exp2 $>$}
\doxylink{class_static}{Static} uninitialized buffer 

One of available type for {\ttfamily \doxylink{structcds_1_1opt_1_1buffer}{opt\+::buffer}} option.

This buffer maintains static array of uninitialized elements. You should manually construct each element when needed. No dynamic memory allocation performed.

\begin{DoxyParagraph}{Template parameters\+:}

\begin{DoxyItemize}
\item {\ttfamily T} -\/ item type the buffer stores
\item {\ttfamily Capacity} -\/ the capacity of buffer. The value must be power of two if {\ttfamily Exp2} is {\ttfamily true} 
\item {\ttfamily Exp2} -\/ a boolean flag. If it is {\ttfamily true} the buffer capacity must be power of two. Otherwise it can be any positive number. Usually, it is required that the buffer has size of a power of two. 
\end{DoxyItemize}
\end{DoxyParagraph}
