\doxysubsection{详细描述}
Atomic item counter 

This class is simplified interface around {\ttfamily std\+::atomic\+\_\+size\+\_\+t}. The class supports getting current value of the counter and increment/decrement its value.

See also\+: improved version that eliminates false sharing -\/ {\ttfamily \doxylink{classcds_1_1atomicity_1_1cache__friendly__item__counter}{cache\+\_\+friendly\+\_\+item\+\_\+counter}}.
