\doxysubsection{详细描述}
Elapsed time timer keeping the best try.

On POSIX platforms we use clock\+\_\+gettime with CLOCK\+\_\+\+PROCESS\+\_\+\+CPUTIME\+\_\+\+ID. On Windows we use Query\+Performance\+Counter

Important\+: on linux, you must link with -\/lrt
