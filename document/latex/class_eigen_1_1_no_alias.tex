\subsubsection*{template$<$typename Expression\+Type, template$<$ typename $>$ class Storage\+Base$>$\newline
class Eigen\+::\+No\+Alias$<$ Expression\+Type, Storage\+Base $>$}
Pseudo expression providing an operator = assuming no aliasing 


\begin{DoxyTemplParams}{模板参数}
{\em Expression\+Type} & the type of the object on which to do the lazy assignment\\
\hline
\end{DoxyTemplParams}
This class represents an expression with special assignment operators assuming no aliasing between the target expression and the source expression. More precisely it alloas to bypass the Eval\+Before\+Assign\+Bit flag of the source expression. It is the return type of \doxylink{class_eigen_1_1_matrix_base_a2c1085de7645f23f240876388457da0b}{Matrix\+Base\+::noalias()} and most of the time this is the only way it is used.
