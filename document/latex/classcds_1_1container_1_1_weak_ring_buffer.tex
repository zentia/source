class cds\+::container\+::\+Weak\+Ring\+Buffer$<$ T, Traits $>$}
Single-\/producer single-\/consumer ring buffer 

Source\+: \mbox{[}2013\mbox{]} Nhat Minh Le, Adrien Guatto, Albert Cohen, Antoniu Pop. Correct and Effcient Bounded FIFO Queues. \mbox{[}Research Report\mbox{]} RR-\/8365, INRIA. 2013. \texorpdfstring{$<$}{<}hal-\/00862450\texorpdfstring{$>$}{>}

Ring buffer is a bounded queue. Additionally, {\ttfamily Weak\+Ring\+Buffer} supports batch operations -\/ you can push/pop an array of elements.

There are a specialization \doxylink{classcds_1_1container_1_1_weak_ring_buffer_3_01void_00_01_traits_01_4_cds_nonintrusive_WeakRingBuffer_void}{Weak\+Ring\+Buffer\texorpdfstring{$<$}{<}void, Traits\texorpdfstring{$>$}{>}} that is not a queue but a "{}memory pool"{} between producer and consumer threads. {\ttfamily Weak\+Ring\+Buffer$<$void$>$} supports variable-\/sized data.

\begin{DoxyWarning}{警告}
\+: {\ttfamily Weak\+Ring\+Buffer} is developed for 64-\/bit architecture. 32-\/bit platform must provide support for 64-\/bit atomics. 
\end{DoxyWarning}
