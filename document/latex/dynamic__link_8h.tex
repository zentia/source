Fill in dynamically linked handlers. 

\textquotesingle{}library\textquotesingle{} is the name of the requested library. It should not contain a full path since dynamic\+\_\+link adds the full path (from which the runtime itself was loaded) to the library name. \textquotesingle{}required\textquotesingle{} is the number of the initial entries in the array descriptors\mbox{[}\mbox{]} that have to be found in order for the call to succeed. If the library and all the required handlers are found, then the corresponding handler pointers are set, and the return value is true. Otherwise the original array of descriptors is left untouched and the return value is false. \textquotesingle{}required\textquotesingle{} is limited by 20 (exceeding of this value will result in failure to load the symbols and the return value will be false). \textquotesingle{}handle\textquotesingle{} is the handle of the library if it is loaded. Otherwise it is left untouched. \textquotesingle{}flags\textquotesingle{} is the set of DYNAMIC\+\_\+\+LINK\+\_\+\texorpdfstring{$\ast$}{*} flags.
