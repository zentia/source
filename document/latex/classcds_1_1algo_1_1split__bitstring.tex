class cds\+::algo\+::split\+\_\+bitstring$<$ Bit\+String, Bit\+String\+Size, UInt $>$}
Cuts a bit sequence from fixed-\/size bit-\/string 

The splitter can be used as an iterator over bit-\/string. Each call of {\ttfamily \doxylink{classcds_1_1algo_1_1split__bitstring_a75a857061ec8d4854305ffab41ce6575}{cut()}} or {\ttfamily \doxylink{classcds_1_1algo_1_1split__bitstring_aa7755ffa786a65c5a028a324bb366562}{safe\+\_\+cut()}} cuts the bit count specified and keeps the position inside bit-\/string for the next call.

The splitter stores a const reference to bit-\/string, not a copy. The maximum count of bits that can be cut in a single call is {\ttfamily  sizeof(\+UInt) \texorpdfstring{$\ast$}{*} 8 }

The splitter keeps byte order.

Template parameters\+:
\begin{DoxyItemize}
\item {\ttfamily Bit\+String} -\/ a fixed-\/sized type that interprets as bit string
\item {\ttfamily Bit\+String\+Size} -\/ the size of {\ttfamily Bit\+String} in bytes, default is {\ttfamily sizeof( Bit\+String )}. You can specify 0 for default.
\item {\ttfamily UInt} -\/ an unsigned integer, return type for {\ttfamily \doxylink{classcds_1_1algo_1_1split__bitstring_a75a857061ec8d4854305ffab41ce6575}{cut()}}, default is {\ttfamily unsigned} 
\end{DoxyItemize}

There are specialized splitters\+:
\begin{DoxyItemize}
\item a simplified {\ttfamily \doxylink{classcds_1_1algo_1_1byte__splitter}{byte\+\_\+splitter}} algorithm that is suitable when count is multiple of 8.
\item {\ttfamily \doxylink{classcds_1_1algo_1_1number__splitter}{number\+\_\+splitter}} algorithm is suitable for a number 
\end{DoxyItemize}
