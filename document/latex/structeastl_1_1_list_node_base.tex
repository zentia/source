\doxysubsection{详细描述}
\doxylink{structeastl_1_1_list_node_base}{List\+Node\+Base}

We define a \doxylink{structeastl_1_1_list_node_base}{List\+Node\+Base} separately from \doxylink{structeastl_1_1_list_node}{List\+Node} (below), because it allows us to have non-\/templated operations such as insert, remove (below), and it makes it so that the list anchor node doesn\textquotesingle{}t carry a T with it, which would waste space and possibly lead to surprising the user due to extra Ts existing that the user didn\textquotesingle{}t explicitly create. The downside to all of this is that it makes debug viewing of a list harder, given that the node pointers are of type \doxylink{structeastl_1_1_list_node_base}{List\+Node\+Base} and not \doxylink{structeastl_1_1_list_node}{List\+Node}. However, see List\+Node\+Base\+Proxy below.
