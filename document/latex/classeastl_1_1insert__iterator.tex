\doxylink{classeastl_1_1insert__iterator}{insert\+\_\+iterator}

An \doxylink{classeastl_1_1insert__iterator}{insert\+\_\+iterator} is like an iterator except that when you assign a value to it, the \doxylink{classeastl_1_1insert__iterator}{insert\+\_\+iterator} inserts the value into the container and increments the iterator.

\doxylink{classeastl_1_1insert__iterator}{insert\+\_\+iterator} is an iterator adaptor that functions as an Output\+Iterator\+: assignment through an \doxylink{classeastl_1_1insert__iterator}{insert\+\_\+iterator} inserts an object into a container. Specifically, if ii is an \doxylink{classeastl_1_1insert__iterator}{insert\+\_\+iterator}, then ii keeps track of a container c and an insertion point p; the expression \texorpdfstring{$\ast$}{*}ii = x performs the insertion container.\+insert(p, x).

If you assign through an \doxylink{classeastl_1_1insert__iterator}{insert\+\_\+iterator} several times, then you will be inserting several elements into the underlying container. In the case of a sequence, they will appear at a particular location in the underlying sequence, in the order in which they were inserted\+: one of the arguments to \doxylink{classeastl_1_1insert__iterator}{insert\+\_\+iterator}\textquotesingle{}s constructor is an iterator p, and the new range will be inserted immediately before p.
