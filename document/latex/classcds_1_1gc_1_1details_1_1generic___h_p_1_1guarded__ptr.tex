Guarded pointer 

\doxylink{bench__gemm_8cpp_addc86e8508f14411ec98f521c520f875}{A} guarded pointer is a pair of a pointer and GC\textquotesingle{}s guard. Usually, it is used for returning a pointer to an element of a lock-\/free container. The guard prevents the pointer to be early disposed (freed) by SMR. After destructing {\ttfamily guarded\+\_\+ptr} object the pointer can be disposed (freed) automatically at any time.

Template arguments\+:
\begin{DoxyItemize}
\item {\ttfamily Guarded\+Type} -\/ a type which the guard stores
\item {\ttfamily \doxylink{test__parallel__reduce_8cpp_aac3e1a5e422a80a86b55a5c15b106ba9}{Value\+Type}} -\/ a value type
\item {\ttfamily Cast} -\/ a functor for converting {\ttfamily Guarded\+Type\texorpdfstring{$\ast$}{*}} to {\ttfamily Value\+Type\texorpdfstring{$\ast$}{*}}. Default is {\ttfamily void} (no casting).
\end{DoxyItemize}

For intrusive containers, {\ttfamily Guarded\+Type} is the same as {\ttfamily \doxylink{test__parallel__reduce_8cpp_aac3e1a5e422a80a86b55a5c15b106ba9}{Value\+Type}} and no casting is needed. In such case the {\ttfamily guarded\+\_\+ptr} is\+: 
\begin{DoxyCode}{0}
\DoxyCodeLine{\textcolor{keyword}{typedef}\ cds::gc::HP::guarded\_ptr<\ foo\ >\ intrusive\_guarded\_ptr;}

\end{DoxyCode}


For standard (non-\/intrusive) containers {\ttfamily Guarded\+Type} is not the same as {\ttfamily \doxylink{test__parallel__reduce_8cpp_aac3e1a5e422a80a86b55a5c15b106ba9}{Value\+Type}} and casting is needed. For example\+:
