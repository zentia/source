\subsubsection*{template$<$typename \+\_\+\+Matrix\+Type, \mbox{\hyperlink{sugar_8h_ae4f82344f573c70aa1066a2394bba345}{int}} \+\_\+\+Up\+Lo = Lower$>$\newline
class Eigen\+::\+Cholmod\+Supernodal\+LLT$<$ \+\_\+\+Matrix\+Type, \+\_\+\+Up\+Lo $>$}
\doxylink{bench__gemm_8cpp_addc86e8508f14411ec98f521c520f875}{A} supernodal Cholesky (\doxylink{class_eigen_1_1_l_l_t}{LLT}) factorization and solver based on Cholmod 

This class allows to solve for A.\+X = \doxylink{bench__gemm_8cpp_a37a83060ac796961b44991c836f083f7}{B} sparse linear problems via a supernodal LL\texorpdfstring{$^\wedge$}{\string^}T Cholesky factorization using the Cholmod library. This supernodal variant performs best on dense enough problems, e.\+g., 3D FEM, or very high order 2D FEM. The sparse matrix \doxylink{bench__gemm_8cpp_addc86e8508f14411ec98f521c520f875}{A} must be selfadjoint and positive definite. The vectors or matrices X and \doxylink{bench__gemm_8cpp_a37a83060ac796961b44991c836f083f7}{B} can be either dense or sparse.


\begin{DoxyTemplParams}{模板参数}
{\em \+\_\+\+Matrix\+Type} & the type of the sparse matrix \doxylink{bench__gemm_8cpp_addc86e8508f14411ec98f521c520f875}{A}, it must be a \doxylink{class_eigen_1_1_sparse_matrix}{Sparse\+Matrix$<$$>$} \\
\hline
{\em \+\_\+\+Up\+Lo} & the triangular part that will be used for the computations. It can be Lower or Upper. Default is Lower.\\
\hline
\end{DoxyTemplParams}
\textbackslash{}implsparsesolverconcept

This class supports all kind of \doxylink{class_eigen_1_1_sparse_matrix}{Sparse\+Matrix$<$$>$}\+: row or column major; upper, lower, or both; compressed or non compressed.

\begin{DoxyWarning}{警告}
Only double precision real and complex scalar types are supported by Cholmod.
\end{DoxyWarning}
