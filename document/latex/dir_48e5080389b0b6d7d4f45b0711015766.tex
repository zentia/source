\begin{DoxyItemize}
\item Very fast (see benchmarks below).
\item Headers only or compiled
\item Feature-\/rich formatting, using the excellent \href{https://github.com/fmtlib/fmt}{\texttt{ fmt}} library.
\item Asynchronous mode (optional)
\item \href{https://github.com/gabime/spdlog/wiki/3.-Custom-formatting}{\texttt{ Custom}} formatting.
\item Multi/\+Single threaded loggers.
\item Various log targets\+:
\begin{DoxyItemize}
\item Rotating log files.
\item Daily log files.
\item Console logging (colors supported).
\item syslog.
\item Windows event log.
\item Windows debugger ({\ttfamily Output\+Debug\+String(..)}).
\item Log to Qt widgets (example).
\item Easily \href{https://github.com/gabime/spdlog/wiki/4.-Sinks\#implementing-your-own-sink}{\texttt{ extendable}} with custom log targets.
\end{DoxyItemize}
\item Log filtering -\/ log levels can be modified at runtime as well as compile time.
\item Support for loading log levels from argv or environment var.
\item Backtrace support -\/ store debug messages in a ring buffer and display them later on demand.
