\doxysubsection{详细描述}
This file contains the definition of the bounded {\ttfamily SPSC} channel buffer used in Fast\+Flow 

Single-\/\+Writer Single-\/\+Reader circular buffer. No lock is needed around pop and push methods. Wait-\/free and fence-\/free (in the TSO model).

\doxylink{bench__gemm_8cpp_addc86e8508f14411ec98f521c520f875}{A} single NULL value is used to indicate buffer full and buffer empty conditions.
