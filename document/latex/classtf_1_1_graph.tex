\doxysubsection{详细描述}
class to create a graph object 

\doxylink{bench__gemm_8cpp_addc86e8508f14411ec98f521c520f875}{A} graph is the ultimate storage for a task dependency graph and is the main gateway to interact with an executor. \doxylink{bench__gemm_8cpp_addc86e8508f14411ec98f521c520f875}{A} graph manages a set of nodes in a global object pool that animates and recycles node objects efficiently without going through repetitive and expensive memory allocations and deallocations. This class is mainly used for creating an opaque graph object in a custom class to interact with the executor through taskflow composition.

\doxylink{bench__gemm_8cpp_addc86e8508f14411ec98f521c520f875}{A} graph object is move-\/only.
