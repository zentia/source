class eastl\+::generic\+\_\+iterator$<$ Iterator, Container $>$}
\doxylink{classeastl_1_1generic__iterator}{generic\+\_\+iterator}

Converts something which can be iterated into a formal iterator. While this class\textquotesingle{} primary purpose is to allow the conversion of a pointer to an iterator, you can convert anything else to an iterator by defining an \doxylink{structeastl_1_1iterator__traits}{iterator\+\_\+traits$<$$>$} specialization for that object type. See EASTL iterator.\+h for this.

Example usage\+: typedef generic\+\_\+iterator$<$int\texorpdfstring{$\ast$}{*}$>$ Int\+Array\+Iterator; typedef generic\+\_\+iterator$<$int\texorpdfstring{$\ast$}{*}, char$>$ Int\+Array\+Iterator\+Other;
