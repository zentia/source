DACFunction it represents the generic (recursive) task of a Divide and Conquer algorithm. Threfore the operand is \textquotesingle{}divided\textquotesingle{}, then it is called the DACFunction for each suboperands and finally the Combine produces the final output. It supports unknown branch factor (i.\+e. unknown number of sub operands). For each recursive call, if n is the branch factor, are created\+: 


\begin{DoxyItemize}
