class cds\+::opt\+::v\+::random\+\_\+shuffle\+\_\+permutation$<$ Int, Random\+Generator, Random\+Device $>$}
Permutation generator based on {\ttfamily std\+::shuffle} 

The class is suitable for {\ttfamily \doxylink{structcds_1_1opt_1_1permutation__generator}{opt\+::permutation\+\_\+generator}} option.

The generator produces a permutation of {\ttfamily \mbox{[}0, n\+Len)} sequence. The generator instance allocates a memory block.

Template parameters\+:
\begin{DoxyItemize}
\item {\ttfamily Int} -\/ specifies the type of generated value, it should be any integer.
\item {\ttfamily Random\+Generator} -\/ random generator, default is {\ttfamily std\+::mt19937} 
\item {\ttfamily Random\+Device} -\/ random device, default is {\ttfamily std\+::random\+\_\+device} 
\end{DoxyItemize}
