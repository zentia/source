\subsubsection*{template$<$typename \+\_\+\+Matrix\+Type, \mbox{\hyperlink{sugar_8h_ae4f82344f573c70aa1066a2394bba345}{int}} \+\_\+\+Up\+Lo, typename \+\_\+\+Ordering$>$\newline
class Eigen\+::\+Simplicial\+LDLT$<$ \+\_\+\+Matrix\+Type, \+\_\+\+Up\+Lo, \+\_\+\+Ordering $>$}
\doxylink{bench__gemm_8cpp_addc86e8508f14411ec98f521c520f875}{A} direct sparse \doxylink{class_eigen_1_1_l_d_l_t}{LDLT} Cholesky factorizations without square root. 

This class provides a LDL\texorpdfstring{$^\wedge$}{\string^}T Cholesky factorizations without square root of sparse matrices that are selfadjoint and positive definite. The factorization allows for solving A.\+X = \doxylink{bench__gemm_8cpp_a37a83060ac796961b44991c836f083f7}{B} where X and \doxylink{bench__gemm_8cpp_a37a83060ac796961b44991c836f083f7}{B} can be either dense or sparse.

In order to reduce the fill-\/in, a symmetric permutation P is applied prior to the factorization such that the factorized matrix is P \doxylink{bench__gemm_8cpp_addc86e8508f14411ec98f521c520f875}{A} P\texorpdfstring{$^\wedge$}{\string^}-\/1.

\hline
