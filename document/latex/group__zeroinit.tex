\doxysubsection{详细描述}

Re-\/allocate memory to {\itshape count} elements of {\itshape size} bytes, with extra memory initialized to zero. 
\begin{DoxyParams}{参数}
{\em p} & Pointer to a previously allocated block (or {\itshape NULL}). \\
\hline
{\em count} & The number of elements. \\
\hline
{\em size} & The size of each element. \\
\hline
\end{DoxyParams}
\begin{DoxyReturn}{返回}
A pointer to a re-\/allocated block of {\itshape count} \texorpdfstring{$\ast$}{*} {\itshape size} bytes, or {\itshape NULL} if out of memory or if {\itshape count} \texorpdfstring{$\ast$}{*} {\itshape size} overflows.
\end{DoxyReturn}
If there is no overflow, it behaves exactly like {\ttfamily mi\+\_\+rezalloc(p,count\texorpdfstring{$\ast$}{*}size)}. \begin{DoxySeeAlso}{参见}
