\doxysubsection{详细描述}
This directory contains source code dedicated to the \`{}xxhsum\`{} command line utility, which is a user program of {\ttfamily libxxhash}.

Note that, in contrast with the library {\ttfamily libxxhash}, the command line utility {\ttfamily xxhsum} ships with GPLv2 license.
