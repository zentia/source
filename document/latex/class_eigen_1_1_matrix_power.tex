\subsubsection*{template$<$typename \mbox{\hyperlink{class_eigen_1_1_transform}{Matrix\+Type}}$>$\newline
class Eigen\+::\+Matrix\+Power$<$ Matrix\+Type $>$}
Class for computing matrix powers. 


\begin{DoxyTemplParams}{模板参数}
{\em \doxylink{class_eigen_1_1_transform}{Matrix\+Type}} & type of the base, expected to be an instantiation of the \doxylink{class_eigen_1_1_matrix}{Matrix} class template.\\
\hline
\end{DoxyTemplParams}
This class is capable of computing real/complex matrices raised to an arbitrary real power. Meanwhile, it saves the result of Schur decomposition if an non-\/integral power has even been calculated. Therefore, if you want to compute multiple (\texorpdfstring{$>$}{>}= 2) matrix powers for the same matrix, using the class directly is more efficient than calling Matrix\+Base\+::pow().
