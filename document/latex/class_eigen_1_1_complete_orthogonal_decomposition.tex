\subsubsection*{template$<$typename \+\_\+\+Matrix\+Type$>$\newline
class Eigen\+::\+Complete\+Orthogonal\+Decomposition$<$ \+\_\+\+Matrix\+Type $>$}
Complete orthogonal decomposition (COD) of a matrix. 


\begin{DoxyParams}{参数}
{\em \doxylink{class_eigen_1_1_complete_orthogonal_decomposition_a8eba7b5263a0241cb477b3531f5ef609}{Matrix\+Type}} & the type of the matrix of which we are computing the COD.\\
\hline
\end{DoxyParams}
This class performs a rank-\/revealing complete orthogonal decomposition of a matrix {\bfseries{\doxylink{bench__gemm_8cpp_addc86e8508f14411ec98f521c520f875}{A}}} into matrices {\bfseries{P}}, {\bfseries{Q}}, {\bfseries{T}}, and {\bfseries{Z}} such that     \[ \mathbf{A} \, \mathbf{P} = \mathbf{Q} \,
                    \begin{bmatrix} \mathbf{T} &  \mathbf{0} \\
                                    \mathbf{0} & \mathbf{0} \end{bmatrix} \, \mathbf{Z}
\] by using Householder transformations. Here, {\bfseries{P}} is a permutation matrix, {\bfseries{Q}} and {\bfseries{Z}} are unitary matrices and {\bfseries{T}} an upper triangular matrix of size rank-\/by-\/rank. {\bfseries{\doxylink{bench__gemm_8cpp_addc86e8508f14411ec98f521c520f875}{A}}} may be rank deficient.

This class supports the \doxylink{group___inplace_decomposition}{inplace decomposition } mechanism.
