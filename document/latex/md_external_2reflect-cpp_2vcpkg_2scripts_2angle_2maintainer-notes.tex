The ANGLE port\textquotesingle{}s buildsystem is based off of \href{https://github.com/WebKit/WebKit/tree/main/Source/ThirdParty/ANGLE}{\texttt{ Web\+Kit\textquotesingle{}s approach}} to converting ANGLE\textquotesingle{}s buildsystem to CMake.

Details\+:


\begin{DoxyItemize}
\item {\ttfamily cmake-\/buildsystem/\+CMake\+Lists.\+txt}
\begin{DoxyItemize}
\item This is an augmented version of Web\+Kit\textquotesingle{}s \href{https://github.com/WebKit/WebKit/blob/main/Source/ThirdParty/ANGLE/CMakeLists.txt}{\texttt{ CMake\+Lists.\+txt}}, with vcpkg edits and additions.
\end{DoxyItemize}
\item {\ttfamily cmake-\/buildsystem/\texorpdfstring{$\ast$}{*}.cmake}
\begin{DoxyItemize}
\item These are configuration files based on \href{https://github.com/WebKit/WebKit/tree/main/Source/ThirdParty/ANGLE}{\texttt{ Web\+Kit\textquotesingle{}s approach}}, but customized for vcpkg (and using {\ttfamily Platform\+Linux} instead of {\ttfamily Platform\+GTK})
\end{DoxyItemize}

\begin{DoxyEnumerate}
\item Select a new ANGLE version
\end{DoxyEnumerate}

Recommendation\+: Follow the recommendations in the lib\+ANGLE "{}\mbox{[}\+Choosing an ANGLE branch\mbox{]}(https\+://github.\+com/google/angle/blob/master/doc/\+Choosing\+ANGLEBranch.\+md\#matching-\/a-\/chromium-\/release-\/to-\/an-\/angle-\/branch)"{} guide, and find the branch that matches the current Chromium Stable.


\begin{DoxyEnumerate}
\item \href{https://github.com/google/angle/branches}{\texttt{ Find the branch}} (usually {\ttfamily chromium/\texorpdfstring{$<$}{<}version\texorpdfstring{$>$}{>}}) and latest commit on that branch, and update the following variables in {\ttfamily portfile.\+cmake}\+:
\begin{DoxyItemize}
\item {\ttfamily ANGLE\+\_\+\+COMMIT}
\item {\ttfamily ANGLE\+\_\+\+VERSION}
\item {\ttfamily ANGLE\+\_\+\+SHA512}
\end{DoxyItemize}
\item \doxylink{struct_check}{Check} the {\ttfamily DEPS} file at that ANGLE commit, for the commit used in `\textquotesingle{}third\+\_\+party/zlib'{\ttfamily , and update}ANGLE\+\_\+\+THIRDPARTY\+\_\+\+ZLIB\+\_\+\+COMMIT{\ttfamily in}portfile.\+cmake\`{} if necessary.
\item Attempt to build. You may have to tweak the {\ttfamily CMake\+Lists.\+txt}, {\ttfamily Platform\texorpdfstring{$\ast$}{*}.cmake} files, etc. \doxylink{struct_check}{Check} with the latest files in \href{https://github.com/WebKit/WebKit/tree/main/Source/ThirdParty/ANGLE}{\texttt{ Web\+Kit\textquotesingle{}s repo}} to see if any updates need to be ported to the vcpkg\textquotesingle{}s custom files. You might also need to change the {\ttfamily ANGLE\+\_\+\+WEBKIT\+\_\+\+BUILDSYSTEM\+\_\+\+COMMIT} in the portfile (although this should theoretically need updating less frequently).
\item \doxylink{struct_check}{Check} headers against {\ttfamily opengl-\/registry} -\/ make sure headers are similar. \begin{quote}
angle defines some additional entrypoints. opengl-\/registry should be latest before updating angle \end{quote}

\end{DoxyEnumerate}
