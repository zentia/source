\doxysubsection{详细描述}
Stores the current pose for a transform constraint. \doxylink{bench__gemm_8cpp_addc86e8508f14411ec98f521c520f875}{A} transform constraint adjusts the world transform of the constrained bones to match that of the target bone. 

See \href{https://esotericsoftware.com/spine-transform-constraints}{\texttt{ Transform constraints}} in the \doxylink{namespace_spine}{Spine} User Guide.
