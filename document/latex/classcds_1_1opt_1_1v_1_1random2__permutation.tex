class cds\+::opt\+::v\+::random2\+\_\+permutation$<$ Int $>$}
Permutation generator of power-\/of-\/2 length based on {\ttfamily rand()} 

The class is suitable for {\ttfamily \doxylink{structcds_1_1opt_1_1permutation__generator}{opt\+::permutation\+\_\+generator}} option.

The generator calculates {\ttfamily n = rand()} and produces the sequence {\ttfamily \mbox{[}n \% n\+Len, (n + 1) \% n\+Len, ..., (n + n\+Len -\/ 1) \% n\+Len\mbox{]}}. The generator does not allocate any memory. {\ttfamily n\+Len} must be power of two.

{\ttfamily Int} template argument specifies the type of generated value, it should be any integer.
