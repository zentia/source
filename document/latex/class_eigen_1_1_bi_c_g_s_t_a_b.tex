\subsubsection*{template$<$typename \+\_\+\+Matrix\+Type, typename \+\_\+\+Preconditioner$>$\newline
class Eigen\+::\+Bi\+CGSTAB$<$ \+\_\+\+Matrix\+Type, \+\_\+\+Preconditioner $>$}
\doxylink{bench__gemm_8cpp_addc86e8508f14411ec98f521c520f875}{A} bi conjugate gradient stabilized solver for sparse square problems 

This class allows to solve for A.\+x = b sparse linear problems using a bi conjugate gradient stabilized algorithm. The vectors x and b can be either dense or sparse.


\begin{DoxyTemplParams}{模板参数}
{\em \+\_\+\+Matrix\+Type} & the type of the sparse matrix \doxylink{bench__gemm_8cpp_addc86e8508f14411ec98f521c520f875}{A}, can be a dense or a sparse matrix. \\
\hline
{\em \+\_\+\+Preconditioner} & the type of the preconditioner. Default is \doxylink{class_eigen_1_1_diagonal_preconditioner}{Diagonal\+Preconditioner}\\
\hline
\end{DoxyTemplParams}
\textbackslash{}implsparsesolverconcept

The maximal number of iterations and tolerance value can be controlled via the \doxylink{class_eigen_1_1_iterative_solver_base_a1bbaf86b7df6d54e78ced4da6ed2dac1}{set\+Max\+Iterations()} and \doxylink{class_eigen_1_1_iterative_solver_base_ac43c50c69a5b621cb5a9991a1005750d}{set\+Tolerance()} methods. The defaults are the size of the problem for the maximal number of iterations and \doxylink{struct_eigen_1_1_generic_num_traits_aadcf7fa75b2bc9a0e461a3b851d836a7}{Num\+Traits$<$\+Scalar$>$\+::epsilon()} for the tolerance.

The tolerance corresponds to the relative residual error\+: \texorpdfstring{$\vert$}{|}\+Ax-\/b\texorpdfstring{$\vert$}{|}/\texorpdfstring{$\vert$}{|}b\texorpdfstring{$\vert$}{|}

{\bfseries{Performance\+:}} when using sparse matrices, best performance is achied for a row-\/major sparse matrix format. Moreover, in this case multi-\/threading can be exploited if the user code is compiled with Open\+MP enabled. See \doxylink{TopicMultiThreading}{Eigen and multi-\/threading} for details.
