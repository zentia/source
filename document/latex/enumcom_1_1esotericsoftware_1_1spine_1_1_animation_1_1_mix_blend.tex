com.\+esotericsoftware.\+spine.\+Animation.\+Mix\+Blend.\+add}

Transitions from the current value to the current value plus the timeline value. No change is made before the first frame (the current value is kept until the first frame). 

{\ttfamily add} is intended for animations layered on top of others, not for the first animations applied. Properties set by additive animations must be set manually or by another animation before applying the additive animations, else the property values will increase each time the additive animations are applied. \Hypertarget{enumcom_1_1esotericsoftware_1_1spine_1_1_animation_1_1_mix_blend_a8c8f885cea5c5ed405aeb571f944c872}\index{com.esotericsoftware.spine.Animation.MixBlend@{com.esotericsoftware.spine.Animation.MixBlend}!first@{first}}
