\doxysection{Release 3.10.0 (Master)}
\hypertarget{release-3-10-0}{}\label{release-3-10-0}\index{Release 3.10.0 (Master)@{Release 3.10.0 (Master)}}
Taskflow 3.\+10.\+0 is the newest developing line to new features and improvements we continue to support. It is also where this documentation is generated. Many things are considered {\itshape experimental} and may change or break from time to time. While it may be difficult to be keep all things consistent when introducing new features, we continue to try our best to ensure backward compatibility.\hypertarget{release-3-10-0_release-3-10-0_download}{}\doxysubsection{\texorpdfstring{Download}{Download}}\label{release-3-10-0_release-3-10-0_download}
To download the newest version of Taskflow, please clone the master branch from \href{https://github.com/taskflow/taskflow}{\texttt{ Taskflow\textquotesingle{}s Git\+Hub}}.\hypertarget{release-3-10-0_release-3-10-0_system_requirements}{}\doxysubsection{\texorpdfstring{System Requirements}{System Requirements}}\label{release-3-10-0_release-3-10-0_system_requirements}
To use Taskflow v3.\+10.\+0, you need a compiler that supports C++17\+:

\begin{DoxyItemize}
\item GNU C++ Compiler at least v8.\+4 with -\/std=c++17 \item Clang C++ Compiler at least v6.\+0 with -\/std=c++17 \item Microsoft Visual Studio at least v19.\+27 with /std\+:c++17 \item Apple\+Clang Xcode Version at least v12.\+0 with -\/std=c++17 \item Nvidia CUDA Toolkit and Compiler (nvcc) at least v11.\+1 with -\/std=c++17 \item Intel C++ Compiler at least v19.\+0.\+1 with -\/std=c++17 \item Intel DPC++ Clang Compiler at least v13.\+0.\+0 with -\/std=c++17\end{DoxyItemize}
Taskflow works on Linux, Windows, and Mac OS X.

\begin{DoxyAttention}{注意}
Although Taskflow supports primarily C++17, you can enable C++20 compilation through {\ttfamily -\/std=c++20} to achieve better performance due to new C++20 features.
\end{DoxyAttention}
