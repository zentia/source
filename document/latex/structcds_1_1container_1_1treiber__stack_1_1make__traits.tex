Metafunction converting option list to {\ttfamily \doxylink{classcds_1_1container_1_1_treiber_stack}{Treiber\+Stack}} traits 

Supported {\ttfamily Options} are\+:
\begin{DoxyItemize}
\item {\ttfamily \doxylink{structcds_1_1opt_1_1allocator}{opt\+::allocator}} -\/ allocator (like {\ttfamily std\+::allocator}) used for allocating stack nodes. Default is \doxylink{external_2libcds_2cds_2user__setup_2allocator_8h_ad5fb7e3cbe8cf24295f0ea3c165a2d82}{CDS\+\_\+\+DEFAULT\+\_\+\+ALLOCATOR}
\item {\ttfamily \doxylink{structcds_1_1opt_1_1back__off}{opt\+::back\+\_\+off}} -\/ back-\/off strategy used. If the option is not specified, the {\ttfamily \doxylink{namespacecds_1_1backoff_a0f64f7bc371e3ad006fba5bd855d6e96}{cds\+::backoff\+::\+Default}} is used.
\item {\ttfamily \doxylink{structcds_1_1opt_1_1memory__model}{opt\+::memory\+\_\+model}} -\/ C++ memory ordering model. Can be {\ttfamily \doxylink{structcds_1_1opt_1_1v_1_1relaxed__ordering}{opt\+::v\+::relaxed\+\_\+ordering}} (relaxed memory model, the default) or {\ttfamily \doxylink{structcds_1_1opt_1_1v_1_1sequential__consistent}{opt\+::v\+::sequential\+\_\+consistent}} (sequentially consisnent memory model).
\item {\ttfamily \doxylink{structcds_1_1opt_1_1item__counter}{opt\+::item\+\_\+counter}} -\/ the type of item counting feature. Default is {\ttfamily \doxylink{classcds_1_1atomicity_1_1empty__item__counter}{cds\+::atomicity\+::empty\+\_\+item\+\_\+counter}}, i.\+e. no item counting. Use {\ttfamily \doxylink{classcds_1_1atomicity_1_1item__counter}{cds\+::atomicity\+::item\+\_\+counter}} to enable item counting.
\item {\ttfamily \doxylink{structcds_1_1opt_1_1stat}{opt\+::stat}} -\/ the type to gather internal statistics. Possible option value are\+: {\ttfamily \doxylink{namespacecds_1_1container_1_1treiber__stack_aefcab8881b6335b606cf738a792a41d4}{treiber\+\_\+stack\+::stat}}, {\ttfamily \doxylink{namespacecds_1_1container_1_1treiber__stack_a4bd0f4032f4015224d068f9a36ec395b}{treiber\+\_\+stack\+::empty\+\_\+stat}} (the default), user-\/provided class that supports {\ttfamily treiber\+\_\+stack\+::stat} interface.
\item {\ttfamily \doxylink{structcds_1_1opt_1_1enable__elimination}{opt\+::enable\+\_\+elimination}} -\/ enable elimination back-\/off for the stack. Default value is {\ttfamily false}.
\end{DoxyItemize}

If elimination back-\/off is enabled, additional options can be specified\+:
\begin{DoxyItemize}
\item {\ttfamily \doxylink{structcds_1_1opt_1_1buffer}{opt\+::buffer}} -\/ an initialized buffer type for elimination array, see {\ttfamily \doxylink{classcds_1_1opt_1_1v_1_1initialized__static__buffer}{opt\+::v\+::initialized\+\_\+static\+\_\+buffer}}, {\ttfamily \doxylink{classcds_1_1opt_1_1v_1_1initialized__dynamic__buffer}{opt\+::v\+::initialized\+\_\+dynamic\+\_\+buffer}}. The buffer can be any size\+: {\ttfamily Exp2} template parameter of those classes can be {\ttfamily false}. The size should be selected empirically for your application and hardware, there are no common rules for that. Default is {\ttfamily  opt\+::v\+::initialized\+\_\+static\+\_\+buffer\texorpdfstring{$<$}{<} \doxylink{structcds_1_1any__type}{any\+\_\+type}, 4 \texorpdfstring{$>$}{>} }.
\item {\ttfamily \doxylink{structcds_1_1opt_1_1random__engine}{opt\+::random\+\_\+engine}} -\/ a random engine to generate a random position in elimination array. Default is {\ttfamily \doxylink{structcds_1_1opt_1_1v_1_1c__rand}{opt\+::v\+::c\+\_\+rand}}.
\item {\ttfamily \doxylink{structcds_1_1opt_1_1elimination__storage}{opt\+::elimination\+\_\+storage}} -\/ a storage class for elmination records. Default is {\ttfamily \doxylink{classcds_1_1algo_1_1elimination_1_1storage}{cds\+::algo\+::elimination\+::storage}}.
\item {\ttfamily \doxylink{structcds_1_1opt_1_1elimination__backoff}{opt\+::elimination\+\_\+backoff}} -\/ back-\/off strategy to wait for elimination, default is {\ttfamily \doxylink{classcds_1_1backoff_1_1delay}{cds\+::backoff\+::delay$<$$>$}} 
\item {\ttfamily \doxylink{structcds_1_1opt_1_1lock__type}{opt\+::lock\+\_\+type}} -\/ a lock type used in elimination back-\/off, default is {\ttfamily \doxylink{namespacecds_1_1sync_a3d645de2cdf4530abac4f1ba0d293ba6}{cds\+::sync\+::spin}}.
\end{DoxyItemize}
