Non-\/resizeable bucket table for {\ttfamily \doxylink{classcds_1_1container_1_1_split_list_set}{Split\+List\+Set}} class. The capacity of table (max bucket count) is defined in the constructor call.

\end{DoxyItemize}

\begin{DoxyItemize}
\item {\ttfamily \doxylink{structcds_1_1opt_1_1allocator}{opt\+::allocator}} -\/ allocator used to allocate bucket table. Default is \doxylink{external_2libcds_2cds_2user__setup_2allocator_8h_ad5fb7e3cbe8cf24295f0ea3c165a2d82}{CDS\+\_\+\+DEFAULT\+\_\+\+ALLOCATOR}
