\doxylink{bench__gemm_8cpp_addc86e8508f14411ec98f521c520f875}{A} sparse LU factorization and solver based on Umf\+Pack 

This class allows to solve for A.\+X = \doxylink{bench__gemm_8cpp_a37a83060ac796961b44991c836f083f7}{B} sparse linear problems via a LU factorization using the Umf\+Pack library. The sparse matrix \doxylink{bench__gemm_8cpp_addc86e8508f14411ec98f521c520f875}{A} must be squared and full rank. The vectors or matrices X and \doxylink{bench__gemm_8cpp_a37a83060ac796961b44991c836f083f7}{B} can be either dense or sparse.

\begin{DoxyWarning}{警告}
The input matrix \doxylink{bench__gemm_8cpp_addc86e8508f14411ec98f521c520f875}{A} should be in a {\bfseries{compressed}} and {\bfseries{column-\/major}} form. Otherwise an expensive copy will be made. You can call the inexpensive \doxylink{_sparse_matrix__coeffs_8cpp_a6086e9aa2a73e5ac235bafa8924f77f8}{make\+Compressed()} to get a compressed matrix. 
\end{DoxyWarning}
