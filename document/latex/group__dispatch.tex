Enables/disables dispatching for AVX2. 

This is automatically detected if it is not defined.
\begin{DoxyItemize}
\item GCC 4.\+7 and later are known to support AVX2, but \texorpdfstring{$>$}{>}4.\+9 is required for to get the AVX2 intrinsics and typedefs without -\/mavx -\/mavx2.
\item Visual Studio 2013 Update 2 and later are known to support AVX2.
\item The GCC/\+Clang internal header {\ttfamily \texorpdfstring{$<$}{<}avx2intrin.\+h\texorpdfstring{$>$}{>}} is detected. While this is not allowed to be included directly, it still appears in the builtin include path and is detectable with {\ttfamily \+\_\+\+\_\+has\+\_\+include}.
\end{DoxyItemize}
