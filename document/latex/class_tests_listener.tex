\doxysubsection{详细描述}
\doxylink{bench__gemm_8cpp_addc86e8508f14411ec98f521c520f875}{A} singleton class. Receives tests results and stores messages to the test log for later listing. It\textquotesingle{}s a singleton for an easy global access from the \textquotesingle{}Asserts\textquotesingle{} methods but it is probably asking for a refactoring in order to limit access only to Test\+Fixtures
