\doxysubsection{详细描述}
The spine-\/sfml runtime provides functionality to load, manipulate and render \href{http://esotericsoftware.com}{\texttt{ Spine}} skeletal animation data using \href{http://www.sfml-dev.org/}{\texttt{ SFML}}. spine-\/sfml is based on \href{../../spine-c}{\texttt{ spine-\/c}}.\hypertarget{README.md_autotoc_md1068}{}\doxysubsection{\texorpdfstring{Licensing}{Licensing}}\label{README.md_autotoc_md1068}

spine-\/sfml works with data exported from \doxylink{namespace_spine}{Spine} 4.\+2.\+xx.

spine-\/sfml supports all \doxylink{namespace_spine}{Spine} features except two color tinting.\hypertarget{README.md_autotoc_md1070}{}\doxysubsection{\texorpdfstring{Usage}{Usage}}\label{README.md_autotoc_md1070}

\begin{DoxyEnumerate}
\item Create a new SFML project. See the \href{http://www.sfml-dev.org/tutorials/2.1/}{\texttt{ SFML documentation}} or have a look at the example in this repository.
\item Download the \doxylink{namespace_spine}{Spine} Runtimes source using git ({\ttfamily git clone \href{https://github.com/esotericsoftware/spine-runtimes}{\texttt{ https\+://github.\+com/esotericsoftware/spine-\/runtimes}}}) or download it as a zip via the download button above.
\item Add the sources from {\ttfamily spine-\/c/spine-\/c/src/spine} and {\ttfamily spine-\/sfml/src/c/spine} to your project
\item Add the folder {\ttfamily spine-\/c/spine-\/c/include} to your header search path. Note that includes are specified as {\ttfamily \#inclue \texorpdfstring{$<$}{<}spine/file.\+h\texorpdfstring{$>$}{>}}, so the {\ttfamily spine} directory cannot be omitted when copying the source files.
\end{DoxyEnumerate}

See the \href{http://esotericsoftware.com/spine-documentation\#runtimesTitle}{\texttt{ Spine Runtimes documentation}} on how to use the APIs or check out the \doxylink{namespace_spine}{Spine} SFML example.\hypertarget{README.md_autotoc_md1071}{}\doxysubsection{\texorpdfstring{Example}{Example}}\label{README.md_autotoc_md1071}
The \doxylink{namespace_spine}{Spine} SFML example works on Windows, Linux and Mac OS X.\hypertarget{README.md_autotoc_md1072}{}\doxysubsubsection{\texorpdfstring{Windows}{Windows}}\label{README.md_autotoc_md1072}

\begin{DoxyEnumerate}
\item Open Visual Studio and open the {\ttfamily spine-\/sfml/c} folder via the {\ttfamily Open a local folder} button

\begin{DoxyEnumerate}
\item Let CMake finish, then select {\ttfamily spine-\/sfml-\/cpp-\/example.\+exe} as the start-\/up project
\end{DoxyEnumerate}
\begin{DoxyEnumerate}
\item Start debugging to run the example
\end{DoxyEnumerate}\hypertarget{README.md_autotoc_md1073}{}\doxysubsubsection{\texorpdfstring{Linux}{Linux}}\label{README.md_autotoc_md1073}

\begin{DoxyEnumerate}

\item \doxylink{ir__common_8h_a1ac75f8fee198af783521627f1fecfbc}{Type} {\ttfamily make -\/j8} to compile the example
