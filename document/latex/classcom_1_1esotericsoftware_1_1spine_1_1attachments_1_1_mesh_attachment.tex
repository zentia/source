\doxysubsection{详细描述}
An attachment that displays a textured mesh. \doxylink{bench__gemm_8cpp_addc86e8508f14411ec98f521c520f875}{A} mesh has hull vertices and internal vertices within the hull. Holes are not supported. Each vertex has UVs (texture coordinates) and triangles are used to map an image on to the mesh. 

See \href{https://esotericsoftware.com/spine-meshes}{\texttt{ Mesh attachments}} in the \doxylink{namespace_spine}{Spine} User Guide.
