class Eigen\+::\+Diagonal$<$ Matrix\+Type, \+\_\+\+Diag\+Index $>$}
Expression of a diagonal/subdiagonal/superdiagonal in a matrix 


\begin{DoxyParams}{参数}
{\em \doxylink{class_eigen_1_1_transform}{Matrix\+Type}} & the type of the object in which we are taking a sub/main/super diagonal \\
\hline
{\em Diag\+Index} & the index of the sub/super diagonal. The default is 0 and it means the main diagonal. \doxylink{bench__gemm_8cpp_addc86e8508f14411ec98f521c520f875}{A} positive value means a superdiagonal, a negative value means a subdiagonal. You can also use Dynamic\+Index so the index can be set at runtime.\\
\hline
\end{DoxyParams}
The matrix is not required to be square.

This class represents an expression of the main diagonal, or any sub/super diagonal of a square matrix. It is the return type of \doxylink{class_eigen_1_1_matrix_base_afe7501ac75694c2056dd24f03de1ad64}{Matrix\+Base\+::diagonal()} and \doxylink{class_eigen_1_1_matrix_base_a8a13d4b8efbd7797ee8efd3dd988a7f7}{Matrix\+Base\+::diagonal(\+Index)} and most of the time this is the only way it is used.
