\subsubsection*{template$<$typename \mbox{\hyperlink{class_eigen_1_1_transform}{Matrix\+Type}}$>$\newline
class Eigen\+::\+Matrix\+Power\+Atomic$<$ Matrix\+Type $>$}
Class for computing matrix powers. 


\begin{DoxyTemplParams}{模板参数}
{\em \doxylink{class_eigen_1_1_transform}{Matrix\+Type}} & type of the base, expected to be an instantiation of the \doxylink{class_eigen_1_1_matrix}{Matrix} class template.\\
\hline
\end{DoxyTemplParams}
This class is capable of computing triangular real/complex matrices raised to a power in the interval $ (-1, 1) $.

\begin{DoxyNote}{注解}
Currently this class is only used by \doxylink{class_eigen_1_1_matrix_power}{Matrix\+Power}. One may insist that this be nested into \doxylink{class_eigen_1_1_matrix_power}{Matrix\+Power}. This class is here to faciliate future development of triangular matrix functions. 
\end{DoxyNote}
