\doxysubsection{详细描述}
Different backoff schemes 

Back-\/off schema may be used in lock-\/free algorithms when the algorithm cannot perform some action because a conflict with the other concurrent operation is encountered. In this case current thread can do another work or can call processor\textquotesingle{}s performance hint.

The interface of back-\/off strategy is following\+:

\DoxyCodeLine{\ \ \ \ \textcolor{keywordtype}{void}\ reset();}
\DoxyCodeLine{\};}

\end{DoxyCode}


{\ttfamily operator()} operator calls back-\/off strategy\textquotesingle{}s action. It is main part of back-\/off strategy.

Interruptible back-\/off {\ttfamily template \texorpdfstring{$<$}{<} typename Predicate \texorpdfstring{$>$}{>} bool operator()( Predicate pr )} allows to interrupt back-\/off spinning if {\ttfamily pr} predicate returns {\ttfamily true}. {\ttfamily Predicate} is a functor with the following interface\+:

{\ttfamily reset()} function resets internal state of back-\/off strategy to initial state. It is required for some back-\/off strategies, for example, exponential back-\/off.
