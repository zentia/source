\doxysubsection{详细描述}
\subsubsection*{template$<$typename Derived$>$\newline
class Eigen\+::\+Eigen\+Base$<$ Derived $>$}
Common base class for all classes T such that \doxylink{class_eigen_1_1_matrix_base}{Matrix\+Base} has an operator=(\+T) and a constructor Matrix\+Base(\+T).

In other words, an \doxylink{struct_eigen_1_1_eigen_base}{Eigen\+Base} object is an object that can be copied into a \doxylink{class_eigen_1_1_matrix_base}{Matrix\+Base}.

Besides Matrix\+Base-\/derived classes, this also includes special matrix classes such as diagonal matrices, etc.

Notice that this class is trivial, it is only used to disambiguate overloaded functions.
