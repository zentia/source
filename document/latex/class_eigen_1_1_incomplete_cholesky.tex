Modified Incomplete Cholesky with dual threshold 

References \+: C-\/J. Lin and J. J. Moré, Incomplete Cholesky Factorizations with Limited memory, SIAM J. Sci. Comput. 21(1), pp. 24-\/45, 1999


\begin{DoxyTemplParams}{模板参数}

\hline
{\em \+\_\+\+Up\+Lo} & The triangular part that will be used for the computations. It can be Lower or Upper. Default is Lower. \\
\hline
{\em \+\_\+\+Ordering\+Type} & The ordering method to use, either \doxylink{class_eigen_1_1_a_m_d_ordering}{AMDOrdering$<$$>$} or \doxylink{class_eigen_1_1_natural_ordering}{Natural\+Ordering$<$$>$}. Default is \doxylink{class_eigen_1_1_a_m_d_ordering}{AMDOrdering$<$int$>$}, unless EIGEN\+\_\+\+MPL2\+\_\+\+ONLY is defined, in which case the default is \doxylink{class_eigen_1_1_natural_ordering}{Natural\+Ordering$<$int$>$}.\\

\textbackslash{}implsparsesolverconcept

It performs the following incomplete factorization\+: $ S P A P\textnormal{\textquotesingle} S \approx L L\textnormal{\textquotesingle} $ where L is a lower triangular factor, S is a diagonal scaling matrix, and P is a fill-\/in reducing permutation as computed by the ordering method.

{\bfseries{Shifting}} {\bfseries{strategy\+:}} Let $ B = S P A P\textnormal{\textquotesingle} S $ be the scaled matrix on which the factorization is carried out, and $ \beta $ be the minimum value of the diagonal. If $ \beta > 0 $ then, the factorization is directly performed on the matrix \doxylink{bench__gemm_8cpp_a37a83060ac796961b44991c836f083f7}{B}. Otherwise, the factorization is performed on the shifted matrix $ B + (\sigma+|\beta| I $ where $ \sigma $ is the initial shift value as returned and set by \doxylink{class_eigen_1_1_incomplete_cholesky_a409c9586e7d29566dda2c8f5e38a1228}{set\+Initial\+Shift()} method. The default value is $ \sigma = 10^{-3} $. If the factorization fails, then the shift in doubled until it succeed or a maximum of ten attempts. If it still fails, as returned by the \doxylink{eigen-3_83_87_2lapack_2cholesky_8cpp_a19e186475eb822b276da4b357d68140d}{info()} method, then you can either increase the initial shift, or better use another preconditioning technique.
