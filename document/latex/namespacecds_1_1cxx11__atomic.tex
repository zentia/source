C++11 Atomic library support 

\label{namespacecds_1_1cxx11__atomic_cds_cxx11_atomic}%
\Hypertarget{namespacecds_1_1cxx11__atomic_cds_cxx11_atomic}%
{\ttfamily libcds} can use the following implementations of the atomics\+:
\begin{DoxyItemize}
\item STL {\ttfamily \texorpdfstring{$<$}{<}atomic\texorpdfstring{$>$}{>}}. This is used by default
\item {\ttfamily boost.\+atomic} for boost 1.\+54 and above. To use it you should define {\ttfamily CDS\+\_\+\+USE\+\_\+\+BOOST\+\_\+\+ATOMIC} for your compiler invocation, for example, for gcc specify {\ttfamily -\/DCDS\+\_\+\+USE\+\_\+\+BOOST\+\_\+\+ATOMIC} in command line
\item {\ttfamily libcds} implementation of atomic operation according to C++11 standard as specified in \href{http://www.open-std.org/jtc1/sc22/wg21/docs/papers/2011/n3242.pdf}{\texttt{ N3242, p.\+29}}. {\ttfamily libcds} implementation is not the full standard compliant, it provides only C++ part of standard, for example, {\ttfamily libcds} has no static initialization of the atomic variables and some other C features. However, that imlementation is enough for the library purposes. Supported architecture\+: x86, amd64, ia64 (Itanium) 64bit, 64bit Sparc. To use {\ttfamily libcds} atomic you should define {\ttfamily CDS\+\_\+\+USE\+\_\+\+LIBCDS\+\_\+\+ATOMIC} in the compiler command line ({\ttfamily -\/DCDS\+\_\+\+USE\+\_\+\+LIBCDS\+\_\+\+ATOMIC} for gcc/clang).

\begin{DoxyNote}{注解}
For Clang compiler {\ttfamily libcds} doesn\textquotesingle{}t use native {\ttfamily libc++} {\ttfamily \texorpdfstring{$<$}{<}atomic\texorpdfstring{$>$}{>}} due some problems. Instead, {\ttfamily libcds} atomic is used by default, or you can try to use {\ttfamily boost.\+atomic}.
\end{DoxyNote}
The library defines {\ttfamily atomics} alias for atomic namespace\+:
\begin{DoxyItemize}
\item {\ttfamily namespace atomics = std} for STL
\item {\ttfamily namespace atomics = boost} for {\ttfamily boost.\+atomic} 
\item {\ttfamily namespace atomics = \doxylink{namespacecds_1_1cxx11__atomic}{cds\+::cxx11\+\_\+atomic}} for library-\/provided atomic implementation 
\end{DoxyItemize}
\end{DoxyItemize}
