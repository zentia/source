\hypertarget{md_external_2reflect-cpp_2docs_2supported__formats_2xml_autotoc_md691}{}\doxysection{\texorpdfstring{Limitations of the XML format}{Limitations of the XML format}}\label{md_external_2reflect-cpp_2docs_2supported__formats_2xml_autotoc_md691}
\hypertarget{md_external_2reflect-cpp_2docs_2supported__formats_2xml_autotoc_md692}{}\doxysubsection{\texorpdfstring{There must be exactly one root node}{There must be exactly one root node}}\label{md_external_2reflect-cpp_2docs_2supported__formats_2xml_autotoc_md692}
The XML format requires that there must be exactly one root node. Other formats, like JSON, allow you to have an array as the root element. XML disallows that.\hypertarget{md_external_2reflect-cpp_2docs_2supported__formats_2xml_autotoc_md693}{}\doxysubsection{\texorpdfstring{The root node must have a name}{The root node must have a name}}\label{md_external_2reflect-cpp_2docs_2supported__formats_2xml_autotoc_md693}
As you may have noticed, reflect-\/cpp uses the names of fields as the names of the XML nodes or attributes. However, the root node does not have a field name associated with it. So reflect-\/cpp will use the name of the struct as the name of the root node. This is a problem when the root node is a template parameter, because the name of the root node cannot contain "{}$<$"{} or "{}$>$"{}.

In this case, you can explicitly set a root node name like this\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{\mbox{\hyperlink{namespacerfl_1_1xml_a903c1af58b9d930d9eed463bc53d2f67}{rfl::xml::write}}<\textcolor{stringliteral}{"{}NameOfTheRootNode"{}}>(homer);}

\end{DoxyCode}
\hypertarget{md_external_2reflect-cpp_2docs_2supported__formats_2xml_autotoc_md694}{}\doxysubsection{\texorpdfstring{No nested arrays}{No nested arrays}}\label{md_external_2reflect-cpp_2docs_2supported__formats_2xml_autotoc_md694}
Unlike JSON, XML doesn\textquotesingle{}t have an explicit concept of arrays. Array-\/like structures are represented by simply writing the same field over and over again (i.\+e. Homer\textquotesingle{}s children in the examples above).
