\doxysubsection{详细描述}
The interface to implement for receiving \doxylink{classcom_1_1esotericsoftware_1_1spine_1_1_animation_state_1_1_track_entry}{Track\+Entry} events. It is always safe to call \doxylink{classcom_1_1esotericsoftware_1_1spine_1_1_animation_state}{Animation\+State} methods when receiving events. 

\doxylink{classcom_1_1esotericsoftware_1_1spine_1_1_animation_state_1_1_track_entry}{Track\+Entry} events are collected during {\ttfamily \doxylink{classcom_1_1esotericsoftware_1_1spine_1_1_animation_state_a11dd94222ec317def80e4b82f78fd1bc}{Animation\+State\#update(float)}} and {\ttfamily \doxylink{classcom_1_1esotericsoftware_1_1spine_1_1_animation_state_acbe6d72550d62c876740b99348a74d39}{Animation\+State\#apply(\+Skeleton)}} and fired only after those methods are finished. 

See \doxylink{classcom_1_1esotericsoftware_1_1spine_1_1_animation_state_1_1_track_entry}{Track\+Entry} {\ttfamily \doxylink{}{Track\+Entry\#set\+Listener(\+Animation\+State\+Listener)}} and \doxylink{classcom_1_1esotericsoftware_1_1spine_1_1_animation_state}{Animation\+State} {\ttfamily \doxylink{classcom_1_1esotericsoftware_1_1spine_1_1_animation_state_a20af544d0cf18ea5165dd035c10bc55f}{Animation\+State\#add\+Listener(\+Animation\+State\+Listener)}}.
