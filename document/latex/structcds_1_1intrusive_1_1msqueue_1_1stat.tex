Queue internal statistics. May be used for debugging or profiling 

Template argument {\ttfamily \doxylink{struct_counter}{Counter}} defines type of counter. Default is {\ttfamily \doxylink{classcds_1_1atomicity_1_1event__counter}{cds\+::atomicity\+::event\+\_\+counter}}, that is weak, i.\+e. it is not guaranteed strict event counting. You may use stronger type of counter like as {\ttfamily \doxylink{classcds_1_1atomicity_1_1item__counter}{cds\+::atomicity\+::item\+\_\+counter}}, or even integral type, for example, {\ttfamily int}.
