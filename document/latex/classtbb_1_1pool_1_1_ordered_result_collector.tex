\doxysubsection{详细描述}
\begin{DoxyVerb}An AbstractResultCollector implementation that collects the
values of the ApplyResult objects in the order they have been
submitted. The CollectorIterator object returned by __iter__()
will iterate over them in the order they have been submitted\end{DoxyVerb}
