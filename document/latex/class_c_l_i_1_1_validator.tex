\doxysubsection{详细描述}
Some validators that are provided 

These are simple {\ttfamily std\+::string(const std\+::string\&)} validators that are useful. They return a string if the validation fails. \doxylink{bench__gemm_8cpp_addc86e8508f14411ec98f521c520f875}{A} custom struct is provided, as well, with the same user semantics, but with the ability to provide a new type name.
