\doxysubsection{详细描述}
class to construct a cu\+BLAS task graph 

cublas\+Flow\+Capturer provides a higher-\/level interface over the @cu\+BLAS library and hide concurrency details from users. It inherits methods from tf\+::cuda\+Flow\+Capturer\+Base and must be used from a \doxylink{classtf_1_1cuda_flow_capturer}{tf\+::cuda\+Flow\+Capturer} object. All pointers used to cublas\+Flow\+Capturer methods must be in GPU memory space or managed (i.\+e., {\ttfamily cuda\+Malloc\+Managed}), including scalars, {\ttfamily alpha} and {\ttfamily beta}, input data and output data pointers. The following example uses {\ttfamily cublas$<$t$>$amax} to find the minimum index of the element of the maximum absolute magnitude in a vector.
