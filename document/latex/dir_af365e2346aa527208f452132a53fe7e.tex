\doxysubsection{详细描述}
The spine-\/c-\/unit-\/tests project is to test the \href{http://esotericsoftware.com}{\texttt{ Spine}} skeletal animation system. It does not perform rendering. It is primarily used for regression testing and leak detection. It is designed to be run from a Continuous Integration server and to passively verify changes automatically on check-\/in.\hypertarget{README.md_autotoc_md938}{}\doxysubsection{\texorpdfstring{Mini CPP Unit Testing}{Mini CPP Unit Testing}}\label{README.md_autotoc_md938}
\href{https://sourceforge.net/p/minicppunit/wiki/Home/}{\texttt{ Mini\+Cpp\+Unit}} is a minimal unit testing framework similar to JUnit. It is used here to avoid large dependancies.

Tests are sorted into Suites, Fixtures and Cases. There is one suite, it contains many fixtures and each fixture contains test cases. To turn off a fixture, edit "{}\+Test\+Options.\+h"{}. To turn off specific test cases, comment out the \doxylink{map__insdel__string_8h_a7966361c046c89b622daa9bd17605675}{TEST\+\_\+\+CASE()} line in the fixture\textquotesingle{}s header.\hypertarget{README.md_autotoc_md939}{}\doxysubsection{\texorpdfstring{Memory Leak Detection}{Memory Leak Detection}}\label{README.md_autotoc_md939}
This project includes a very minimal memory leak detector. It is based roughly on the leak detector in the \href{https://sourceforge.net/projects/popcapframework/?source=directory}{\texttt{ Popcap Framework}}, but has been modified over the years.\hypertarget{README.md_autotoc_md940}{}\doxysubsection{\texorpdfstring{Continuous Integration}{Continuous Integration}}\label{README.md_autotoc_md940}
The test runner includes the ability to format output messages to signal a CI server. An example interface for \href{https://www.jetbrains.com/teamcity/}{\texttt{ Teamcity}} is included. To implement for another server, determine the wireformat for the messages and duplicate/edit the teamcity\+\_\+messages class. \href{https://confluence.jetbrains.com/display/TCD10/Build+Script+Interaction+with+TeamCity}{\texttt{ Teamcity Wire Format}}\hypertarget{README.md_autotoc_md941}{}\doxysubsubsection{\texorpdfstring{Trigger}{Trigger}}\label{README.md_autotoc_md941}
