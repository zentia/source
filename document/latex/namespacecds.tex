Initialize CDS library 

The function initializes {\ttfamily CDS} library framework. Before usage of {\ttfamily CDS} library features your application must initialize it by calling {\ttfamily Initialize}() function, see \doxylink{defs.h_cds_how_to_use}{how to use the library}.

You can call {\ttfamily Initialize} several times, only first call is significant others will be ignored. To terminate the {\ttfamily CDS} library correctly, each call to {\ttfamily Initialize}() must be balanced by a corresponding {\ttfamily \doxylink{namespacecds_a9ca275d587d2b0b11ded25220718a049}{Terminate()}} call.

Note, that this function does not initialize garbage collectors. To use GC you need you should call GC-\/specific constructor function to initialize internal structures of GC. See {\ttfamily \doxylink{namespacecds_1_1gc}{cds\+::gc}} for details.
