Transforms the attachment\textquotesingle{}s local \doxylink{class_spine_1_1_vertex_attachment_a1eb0b1000871b10626c168cd52c25447}{Vertices} to world coordinates. If the slot\textquotesingle{}s \doxylink{class_spine_1_1_slot_a5be59110ff332b43a55dfe169770f343}{Slot.\+Deform} is not empty, it is used to deform the vertices. 

See \href{http://esotericsoftware.com/spine-runtime-skeletons\#World-transforms}{\texttt{ World transforms}} in the \doxylink{namespace_spine}{Spine} Runtimes Guide.


\begin{DoxyParams}{参数}
{\em start} & The index of the first \doxylink{class_spine_1_1_vertex_attachment_a1eb0b1000871b10626c168cd52c25447}{Vertices} value to transform. Each vertex has 2 values, x and y.\\
\hline
{\em count} & The number of world vertex values to output. Must be less than or equal to \doxylink{class_spine_1_1_vertex_attachment_a6ccccde921aef221c82b4a71a70fcdad}{World\+Vertices\+Length} -\/ start.\\
\hline
{\em world\+Vertices} & The output world vertices. Must have a length greater than or equal to {\itshape offset}  + {\itshape count} .\\
\hline
{\em offset} & The {\itshape world\+Vertices}  index to begin writing values.\\
\hline
{\em stride} & The number of {\itshape world\+Vertices}  entries between the value pairs written.\\
\hline
