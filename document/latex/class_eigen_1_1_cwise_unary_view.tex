Generic lvalue expression of a coefficient-\/wise unary operator of a matrix or a vector 


\begin{DoxyTemplParams}{模板参数}

This class represents a lvalue expression of a generic unary view operator of a matrix or a vector. It is the return type of \doxylink{datatypes_8h_a1bb7bda9eb99a2427c358ecb686be221}{real()} and \doxylink{namespace_eigen_a524f2d0ae40cbbb4f55ea073c98a1c43}{imag()}, and most of the time this is the only way it is used.
