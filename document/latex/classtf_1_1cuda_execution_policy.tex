\subsubsection*{template$<$unsigned NT, unsigned VT$>$\newline
class tf\+::cuda\+Execution\+Policy$<$ NT, VT $>$}
class to define execution policy for CUDA standard algorithms 


\begin{DoxyTemplParams}{模板参数}
{\em NT} & number of threads per block \\
\hline
{\em VT} & number of work units per thread\\
\hline
\end{DoxyTemplParams}
Execution policy configures the kernel execution parameters in CUDA algorithms. The first template argument, {\ttfamily NT}, the number of threads per block should always be a power-\/of-\/two number. The second template argument, {\ttfamily VT}, the number of work units per thread is recommended to be an odd number to avoid bank conflict.

Details can be referred to \doxylink{CUDASTDExecutionPolicy}{Execution Policy}.
