This is a base class for LL\texorpdfstring{$^\wedge$}{\string^}T and LDL\texorpdfstring{$^\wedge$}{\string^}T Cholesky factorizations of sparse matrices that are selfadjoint and positive definite. These factorizations allow for solving A.\+X = \doxylink{bench__gemm_8cpp_a37a83060ac796961b44991c836f083f7}{B} where X and \doxylink{bench__gemm_8cpp_a37a83060ac796961b44991c836f083f7}{B} can be either dense or sparse.

In order to reduce the fill-\/in, a symmetric permutation P is applied prior to the factorization such that the factorized matrix is P \doxylink{bench__gemm_8cpp_addc86e8508f14411ec98f521c520f875}{A} P\texorpdfstring{$^\wedge$}{\string^}-\/1.
