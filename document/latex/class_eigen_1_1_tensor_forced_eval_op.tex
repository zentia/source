class Eigen\+::\+Tensor\+Forced\+Eval\+Op$<$ Xpr\+Type, Make\+Pointer\+\_\+ $>$}
\doxylink{class_eigen_1_1_tensor}{Tensor} reshaping class. 

{\ttfamily template \texorpdfstring{$<$}{<}class\texorpdfstring{$>$}{>} class Make\+Pointer\+\_\+} is added to convert the host pointer to the device pointer. It is added due to the fact that for our device compiler {\ttfamily T\texorpdfstring{$\ast$}{*}} is not allowed. If we wanted to use the same Evaluator functions we have to convert that type to our pointer {\ttfamily T}. This is done through our {\ttfamily Make\+Pointer\+\_\+} class. By default the \doxylink{ir__common_8h_a1ac75f8fee198af783521627f1fecfbc}{Type} in the {\ttfamily Make\+Pointer\+\_\+\texorpdfstring{$<$}{<}T\texorpdfstring{$>$}{>}} is {\ttfamily T\texorpdfstring{$\ast$}{*}} . Therefore, by adding the default value, we managed to convert the type and it does not break any existing code as its default value is {\ttfamily T\texorpdfstring{$\ast$}{*}}.
