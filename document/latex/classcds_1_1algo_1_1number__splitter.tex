\subsubsection*{template$<$typename Int$>$\newline
class cds\+::algo\+::number\+\_\+splitter$<$ Int $>$}
Cuts a bit sequence from a number 

The splitter can be used as an iterator over bit representation of the number of type {\ttfamily Int}. Each call of {\ttfamily \doxylink{classcds_1_1algo_1_1number__splitter_ae9bf1d402ba0a82e7633e43e9ec57e8f}{cut()}} or {\ttfamily \doxylink{classcds_1_1algo_1_1number__splitter_a2b4a0558e5e9a4e8f8cabafe4c71ff2b}{safe\+\_\+cut()}} cuts the bit count specified and keeps the position inside the number for the next call.
