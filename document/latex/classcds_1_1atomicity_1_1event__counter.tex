\doxysubsection{详细描述}
Atomic event counter. 

This class is based on {\ttfamily std\+::atomic\+\_\+size\+\_\+t}. It uses relaxed memory ordering {\ttfamily memory\+\_\+order\+\_\+relaxed} and may be used as a statistic counter.
