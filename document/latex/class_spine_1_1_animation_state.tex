\doxysubsection{详细描述}
Applies animations over time, queues animations for later playback, mixes (crossfading) between animations, and applies multiple animations on top of each other (layering).

See \href{http://esotericsoftware.com/spine-applying-animations/}{\texttt{ Applying Animations}} in the \doxylink{namespace_spine}{Spine} Runtimes Guide.

Mixing out is done by setting an empty animation with a mix duration using either \doxylink{class_spine_1_1_animation_state_a4e84d7e2712e10ea6ed95ff092cbd379}{Animation\+State.\+Set\+Empty\+Animation(int, float)}, \doxylink{class_spine_1_1_animation_state_a862c95ea332e047c49464ee3514d57f7}{Animation\+State.\+Set\+Empty\+Animations(float)}, or \doxylink{class_spine_1_1_animation_state_ad66c903cef5ad7fa9eee840342d7388a}{Animation\+State.\+Add\+Empty\+Animation(int, float, float)}. Mixing to an empty animation causes the previous animation to be applied less and less over the mix duration. Properties keyed in the previous animation transition to the value from lower tracks or to the setup pose value if no lower tracks key the property. \doxylink{bench__gemm_8cpp_addc86e8508f14411ec98f521c520f875}{A} mix duration of 0 still mixes out over one frame.
