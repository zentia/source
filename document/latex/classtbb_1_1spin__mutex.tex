\doxysubsection{详细描述}
\doxylink{bench__gemm_8cpp_addc86e8508f14411ec98f521c520f875}{A} lock that occupies a single byte. 

\doxylink{bench__gemm_8cpp_addc86e8508f14411ec98f521c520f875}{A} \doxylink{classtbb_1_1spin__mutex}{spin\+\_\+mutex} is a spin mutex that fits in a single byte. It should be used only for locking short critical sections (typically less than 20 instructions) when fairness is not an issue. If zero-\/initialized, the mutex is considered unheld.
