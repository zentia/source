Security updates are applied only to the latest release.\hypertarget{md_external_2magic__enum_2_s_e_c_u_r_i_t_y_autotoc_md515}{}\doxysection{\texorpdfstring{Reporting a Vulnerability}{Reporting a Vulnerability}}\label{md_external_2magic__enum_2_s_e_c_u_r_i_t_y_autotoc_md515}
If you have discovered a security vulnerability in this project, please report it privately. {\bfseries{Do not disclose it as a public issue.}} This gives us time to work with you to fix the issue before public exposure, reducing the chance that the exploit will be used before a patch is released.

Please disclose it at \href{https://github.com/Neargye/magic_enum/security/advisories/new}{\texttt{ security advisory}}.

This project is maintained by a team of volunteers on a reasonable-\/effort basis. As such, vulnerabilities will be disclosed in a best effort base.
