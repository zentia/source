\doxysubsection{详细描述}
Automatic dispatcher code for the \doxylink{group___x_x_h3__family}{XXH3 family} on x86-\/based targets.

Optional add-\/on.

{\bfseries{Compile this file with the default flags for your target.}} Note that compiling with flags like {\ttfamily -\/mavx\texorpdfstring{$\ast$}{*}}, {\ttfamily -\/march=native}, or {\ttfamily /arch\+:AVX\texorpdfstring{$\ast$}{*}} will make the resulting binary incompatible with cpus not supporting the requested instruction set.
