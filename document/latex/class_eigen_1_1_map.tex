This class represents a matrix or vector expression mapping an existing array of data. It can be used to let \doxylink{namespace_eigen}{Eigen} interface without any overhead with non-\/\+Eigen data structures, such as plain C arrays or structures from other libraries. By default, it assumes that the data is laid out contiguously in memory. You can however override this by explicitly specifying inner and outer strides.
