\doxysubsection{详细描述}
Simple yet efficient (not concurrent) unbounded FIFO queue. 

This queue have to be used by one single thread at a time, or, each method call have to be protected by a mutex lock. For an unbounded producer-\/consumer queue implementation see \doxylink{ubuffer_8hpp}{ubuffer.\+hpp}
