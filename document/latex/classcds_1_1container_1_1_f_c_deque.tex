Flat-\/combining deque 

\doxylink{namespacecds_1_1algo_1_1flat__combining_cds_flat_combining_description}{Flat combining} sequential deque. The class can be considered as a concurrent FC-\/based wrapper for {\ttfamily std\+::deque}.

Template parameters\+:
\begin{DoxyItemize}
\item {\ttfamily T} -\/ a value type stored in the deque
\item {\ttfamily Deque} -\/ sequential deque implementation, for example, {\ttfamily std\+::deque$<$\+T$>$} (the default) or {\ttfamily boost\+::container\+::deque} 
\item {\ttfamily Trats} -\/ type traits of flat combining, default is {\ttfamily \doxylink{structcds_1_1container_1_1fcdeque_1_1traits}{fcdeque\+::traits}}. {\ttfamily \doxylink{structcds_1_1container_1_1fcdeque_1_1make__traits}{fcdeque\+::make\+\_\+traits}} metafunction can be used to construct specialized {\ttfamily fcdeque\+::traits}
