class cds\+::urcu\+::general\+\_\+buffered$<$ Buffer, Lock, Backoff $>$}
User-\/space general-\/purpose RCU with deferred (buffered) reclamation 

This URCU implementation contains an internal buffer where retired objects are accumulated. When the buffer becomes full, the RCU {\ttfamily synchronize} function is called that waits until all reader/updater threads end up their read-\/side critical sections, i.\+e. until the RCU quiescent state will come. After that the buffer and all retired objects are freed. This synchronization cycle may be called in any thread that calls {\ttfamily retire\+\_\+ptr} function.

The {\ttfamily \doxylink{struct_buffer}{Buffer}} contains items of \doxylink{namespacecds_1_1urcu_cds_urcu_retired_ptr}{epoch\+\_\+retired\+\_\+ptr} type and it should support a queue interface with three function\+:
\begin{DoxyItemize}
\item {\ttfamily  bool push( retired\+\_\+ptr\& p ) } -\/ places the retired pointer {\ttfamily p} into queue. If the function returns {\ttfamily false} it means that the buffer is full and RCU synchronization cycle must be processed.
\item {\ttfamily bool pop( retired\+\_\+ptr\& p ) } -\/ pops queue\textquotesingle{}s head item into {\ttfamily p} parameter; if the queue is empty this function must return {\ttfamily false} 
\item {\ttfamily size\+\_\+t \doxylink{imgui__impl__opengl3__loader_8h_ac42c05b5701aef6bab82675cf3b55fee}{size()}} -\/ returns queue\textquotesingle{}s item count.
\end{DoxyItemize}

The buffer is considered as full if {\ttfamily \doxylink{test__concurrent__queue_8cpp_a262227aa72b9cc1db7b9a80d7f295211}{push()}} returns {\ttfamily false} or the buffer size reaches the RCU threshold.

There is a wrapper \doxylink{classcds_1_1urcu_1_1gc_3_01general__buffered_3_01_buffer_00_01_lock_00_01_backoff_01_4_01_4_cds_urcu_general_buffered_gc}{gc\texorpdfstring{$<$}{<}general\+\_\+buffered\texorpdfstring{$>$}{>}} for {\ttfamily general\+\_\+buffered} class that provides unified RCU interface. You should use this wrapper class instead {\ttfamily general\+\_\+buffered} 

Template arguments\+:
\begin{DoxyItemize}
\item {\ttfamily \doxylink{struct_buffer}{Buffer}} -\/ buffer type. Default is {\ttfamily \doxylink{classcds_1_1container_1_1_vyukov_m_p_m_c_cycle_queue}{cds\+::container\+::\+Vyukov\+MPMCCycle\+Queue}} 
\item {\ttfamily Lock} -\/ mutex type, default is {\ttfamily std\+::mutex} 
\item {\ttfamily Backoff} -\/ back-\/off schema, default is \doxylink{namespacecds_1_1backoff_a0f64f7bc371e3ad006fba5bd855d6e96}{cds\+::backoff\+::\+Default} 
\end{DoxyItemize}
