\doxysubsection{详细描述}
An attachment which is a single point and a rotation. This can be used to spawn projectiles, particles, etc. \doxylink{bench__gemm_8cpp_addc86e8508f14411ec98f521c520f875}{A} bone can be used in similar ways, but a \doxylink{class_spine_1_1_point_attachment}{Point\+Attachment} is slightly less expensive to compute and can be hidden, shown, and placed in a skin. 

See \href{http://esotericsoftware.com/spine-point-attachments}{\texttt{ Point Attachments}} in the \doxylink{namespace_spine}{Spine} User Guide.
