\doxysubsection{详细描述}
class to create a worker in an executor 

The class is primarily used by the executor to perform work-\/stealing algorithm. Users can access a worker object and alter its property (e.\+g., changing the thread affinity in a POSIX-\/like system) using \doxylink{classtf_1_1_worker_interface}{tf\+::\+Worker\+Interface}.
