JSON Pointer 

\doxylink{bench__gemm_8cpp_addc86e8508f14411ec98f521c520f875}{A} JSON pointer defines a string syntax for identifying a specific value within a JSON document. It can be used with functions {\ttfamily at} and {\ttfamily operator\mbox{[}\mbox{]}}. Furthermore, JSON pointers are the base for JSON patches.
