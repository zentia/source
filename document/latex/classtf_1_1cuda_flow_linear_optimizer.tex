\doxysubsection{详细描述}
class to capture a linear CUDA graph using a sequential stream 

\doxylink{bench__gemm_8cpp_addc86e8508f14411ec98f521c520f875}{A} linear capturing algorithm is a special case of \doxylink{classtf_1_1cuda_flow_sequential_optimizer}{tf\+::cuda\+Flow\+Sequential\+Optimizer} and assumes the input task graph to be a single linear chain of tasks (i.\+e., a straight line). This assumption allows faster optimization during the capturing process. If the input task graph is not a linear chain, the behavior is undefined.
