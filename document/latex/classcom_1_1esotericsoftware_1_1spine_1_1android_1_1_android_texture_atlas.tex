\doxysubsection{详细描述}
Atlas data loaded from a {\ttfamily .atlas} file and its corresponding {\ttfamily .png} files. For each atlas image, a corresponding {\ttfamily \doxylink{}{Bitmap}} and {\ttfamily \doxylink{}{Paint}} is constructed, which are used when rendering a skeleton that uses this atlas.

Use the static methods {\ttfamily \doxylink{classcom_1_1esotericsoftware_1_1spine_1_1android_1_1_android_texture_atlas_a67400b9675b2ed86121b5d7586f15d6d}{Android\+Texture\+Atlas\#from\+Asset(\+String, Context)}}, {\ttfamily \doxylink{classcom_1_1esotericsoftware_1_1spine_1_1android_1_1_android_texture_atlas_a66c576f40e82ec81e637553f1f06e694}{Android\+Texture\+Atlas\#from\+File(\+File)}}, and {\ttfamily \doxylink{classcom_1_1esotericsoftware_1_1spine_1_1android_1_1_android_texture_atlas_a6b39955009b92f2a4c391716a191163b}{Android\+Texture\+Atlas\#from\+Http(\+URL, File)}} to load an atlas.
