class cds\+::details\+::\+Aligned\+Allocator$<$ T, ALIGNED\+\_\+\+ALLOCATOR $>$}
\doxylink{classcds_1_1details_1_1_allocator}{Allocator} for aligned data 

The class is the wrapper around user-\/defined aligned allocator. Template parameters\+: \begin{DoxyItemize}
\item {\ttfamily T} is a type to allocate \item {\ttfamily ALIGNED\+\_\+\+ALLOCATOR} is an aligned allocator implementation. Default implementation is defined by macro CDS\+\_\+\+DEFAULT\+\_\+\+ALIGNED\+\_\+\+ALLOCATOR from \doxylink{external_2libcds_2cds_2user__setup_2allocator_8h}{cds/user\+\_\+setup/allocator.\+h} header file.\end{DoxyItemize}
The {\ttfamily n\+Align} parameter of member function specifyes desired aligment of data allocated.

\begin{DoxyParagraph}{Note}
When an array allocation is performed the allocator guarantees the alignment for first element of array only. To guarantee the alignment for each element of the array the size of type {\ttfamily T} must be multiple of {\ttfamily n\+Align\+:}
