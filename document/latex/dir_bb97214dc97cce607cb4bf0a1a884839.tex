\doxysubsection{详细描述}
The spine-\/csharp runtime provides functionality to load and manipulate \href{http://esotericsoftware.com}{\texttt{ Spine}} skeletal animation data using C\# (C Sharp). It does not perform rendering but can be extended to enable \doxylink{namespace_spine}{Spine} animations for other C\#-\/based projects.\hypertarget{README.md_autotoc_md962}{}\doxysubsection{\texorpdfstring{Licensing}{Licensing}}\label{README.md_autotoc_md962}
