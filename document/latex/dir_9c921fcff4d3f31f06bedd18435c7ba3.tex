\doxysubsection{详细描述}
This directory is meant to provide the sources for the official \href{https://github.com/microsoft/vcpkg/tree/master/ports/mimalloc}{\texttt{ vcpkg port}} of mimalloc, but can also be used to override the official port with your own variant.

For example, you can edit the \href{portfile.cmake}{\texttt{ {\ttfamily portfile.\+cmake}}} to check out a specific commit, version, or branch of mimalloc, or set further options. You can install such custom port as\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{\$\ vcpkg\ install\ "{}mimalloc[override]"{}\ -\/-\/recurse\ -\/-\/overlay-\/ports=./contrib/vcpkg}

\end{DoxyCode}


This will also show the correct sha512 hash if you use a custom version. Another way is to refer to the overlay from the \href{https://learn.microsoft.com/en-us/vcpkg/reference/vcpkg-configuration-json}{\texttt{ vcpkg-\/configuration.\+json}} file. See also the vcpkg \href{https://learn.microsoft.com/en-us/vcpkg/produce/update-package-version}{\texttt{ documentation}} for more information.\hypertarget{readme.md_autotoc_md17}{}\doxysubsection{\texorpdfstring{Using mimalloc from vcpkg}{Using mimalloc from vcpkg}}\label{readme.md_autotoc_md17}
When using \href{https://learn.microsoft.com/en-us/vcpkg/get_started/get-started?pivots=shell-powershell}{\texttt{ cmake with vcpkg}}, you can use mimalloc from the {\ttfamily CMake\+Lists.\+txt} as\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{find\_package(mimalloc\ CONFIG\ REQUIRED)}
\DoxyCodeLine{target\_link\_libraries(main\ PRIVATE\ mimalloc)}

\end{DoxyCode}
