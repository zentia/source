Expression of a triangular part in a matrix 


\begin{DoxyParams}{参数}
{\em \doxylink{class_eigen_1_1_triangular_view_add702d94e40cc9a747412cbce9fc4449}{Matrix\+Type}} & the type of the object in which we are taking the triangular part \\
\hline
{\em \doxylink{monitors_8c_a46c8a310cf4c094f8c80e1cb8dc1f911}{Mode}} & the kind of triangular matrix expression to construct. Can be \doxylink{group__enums_gga39e3366ff5554d731e7dc8bb642f83cdafca2ccebb604f171656deb53e8c083c1}{Upper}, \doxylink{group__enums_gga39e3366ff5554d731e7dc8bb642f83cdaf581029282d421eee5aae14238c6f749}{Lower}, \doxylink{group__enums_gga39e3366ff5554d731e7dc8bb642f83cdadd28224d7ea92689930be73c1b50b0ad}{Unit\+Upper}, \doxylink{group__enums_gga39e3366ff5554d731e7dc8bb642f83cda8f40b928c10a71ba03e5f75ad2a72fda}{Unit\+Lower}, \doxylink{group__enums_gga39e3366ff5554d731e7dc8bb642f83cda7b37877e0b9b0df28c9c2b669a633265}{Strictly\+Upper}, or \doxylink{group__enums_gga39e3366ff5554d731e7dc8bb642f83cda2424988b6fca98be70b595632753ba81}{Strictly\+Lower}. This is in fact a bit field; it must have either \doxylink{group__enums_gga39e3366ff5554d731e7dc8bb642f83cdafca2ccebb604f171656deb53e8c083c1}{Upper} or \doxylink{group__enums_gga39e3366ff5554d731e7dc8bb642f83cdaf581029282d421eee5aae14238c6f749}{Lower}, and additionally it may have \doxylink{group__enums_gga39e3366ff5554d731e7dc8bb642f83cda2ef430bff6cc12c2d1e0ef01b95f7ff3}{Unit\+Diag} or \doxylink{group__enums_gga39e3366ff5554d731e7dc8bb642f83cdac4dc554a61510151ddd5bafaf6040223}{Zero\+Diag} or neither.\\
\hline
\end{DoxyParams}
This class represents a triangular part of a matrix, not necessarily square. Strictly speaking, for rectangular matrices one should speak of "{}trapezoid"{} parts. This class is the return type of \doxylink{class_eigen_1_1_matrix_base_a766e24ff164cf92d9a43702a7accdda3}{Matrix\+Base\+::triangular\+View()} and \doxylink{class_eigen_1_1_sparse_matrix_base_a6762232f126791a057dfbf3bf6014542}{Sparse\+Matrix\+Base\+::triangular\+View()}, and most of the time this is the only way it is used.
