\doxysubsection{详细描述}
Stores the current pose for a path constraint. \doxylink{bench__gemm_8cpp_addc86e8508f14411ec98f521c520f875}{A} path constraint adjusts the rotation, translation, and scale of the constrained bones so they follow a \doxylink{class_spine_1_1_path_attachment}{Path\+Attachment}.

See \href{http://esotericsoftware.com/spine-path-constraints}{\texttt{ Path constraints}} in the \doxylink{namespace_spine}{Spine} User Guide.
