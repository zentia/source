Key size option 

By default, the size of hash value is {\ttfamily sizeof( hash\+\_\+type )} where {\ttfamily hash\+\_\+type} is type of {\ttfamily \doxylink{structhash}{hash()}} result or {\ttfamily sizeof( key )} if you use fixed-\/sized key.

Sometimes that size is wrong, for example, for that 6-\/byte key\+:
