\subsubsection*{template$<$typename Derived$>$\newline
class Eigen\+::\+Map\+Base$<$ Derived, Write\+Accessors $>$}
\doxylink{class_base}{Base} class for non-\/const dense \doxylink{class_eigen_1_1_map}{Map} and \doxylink{class_eigen_1_1_block}{Block} expression with direct access 

This base class provides the non-\/const low-\/level accessors (e.\+g. coeff and coeff\+Ref) of dense \doxylink{class_eigen_1_1_map}{Map} and \doxylink{class_eigen_1_1_block}{Block} objects with direct access. It inherits Map\+Base$<$\+Derived, Read\+Only\+Accessors$>$ which defines the const variant for reading specific entries.
