class Eigen\+::\+Spline$<$ \+\_\+\+Scalar, \+\_\+\+Dim, \+\_\+\+Degree $>$}
\doxylink{bench__gemm_8cpp_addc86e8508f14411ec98f521c520f875}{A} class representing multi-\/dimensional spline curves. 

The class represents B-\/splines with non-\/uniform knot vectors. Each control point of the B-\/spline is associated with a basis function   \begin{align*}  C(u) & = \sum_{i=0}^{n}N_{i,p}(u)P_i
\end{align*}


\begin{DoxyTemplParams}{模板参数}
{\em \+\_\+\+Scalar} & The underlying data type (typically float or double) \\
\hline
{\em \+\_\+\+Dim} & The curve dimension (e.\+g. 2 or 3) \\
\hline
{\em \+\_\+\+Degree} & Per default set to Dynamic; could be set to the actual desired degree for optimization purposes (would result in stack allocation of several temporary variables). \\
\hline
\end{DoxyTemplParams}
