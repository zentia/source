\doxysubsection{详细描述}
Contributed by Todd\+Rivers\hypertarget{README.md_autotoc_md1113}{}\doxysubsection{\texorpdfstring{Unity Sprite Shaders}{Unity Sprite Shaders}}\label{README.md_autotoc_md1113}
An Uber Shader specialised for rendering Sprites in Unity. Even though it\textquotesingle{}s designed for Sprites it can be used for a whole range of uses. It supports a wide range of optional shader features that won\textquotesingle{}t effect performance unless they are used. It also supports per-\/pixel effects such as normal maps and diffuse ramping whilst using \doxylink{struct_vertex}{Vertex} Lit rendering.\hypertarget{README.md_autotoc_md1114}{}\doxysubsubsubsection{\texorpdfstring{Lighting}{Lighting}}\label{README.md_autotoc_md1114}
The shaders support lighting using both Forward Rendering and \doxylink{struct_vertex}{Vertex} Lit Rendering. Forward rendering is more accurate but is slower and crucially means the sprite has to write to depth using alpha clipping to avoid overdraw. \doxylink{struct_vertex}{Vertex} lit means all lighting can be done in one pass meaning full alpha can be used.\hypertarget{README.md_autotoc_md1115}{}\doxysubsubsubsection{\texorpdfstring{Normal Mapping}{Normal Mapping}}\label{README.md_autotoc_md1115}
Normals maps are supported in both lighting modes (in \doxylink{struct_vertex}{Vertex} Lit rendering data for normal mapping is packed into texture channels and then processed per pixel).\hypertarget{README.md_autotoc_md1116}{}\doxysubsubsubsection{\texorpdfstring{Blend Modes}{Blend Modes}}\label{README.md_autotoc_md1116}
Easily switch between blend modes including pre-\/multiplied alpha, additive, multiply etc.\hypertarget{README.md_autotoc_md1117}{}\doxysubsubsubsection{\texorpdfstring{Rim Lighting}{Rim Lighting}}\label{README.md_autotoc_md1117}
Camera-\/space rim lighting is supported in both lighting modes.\hypertarget{README.md_autotoc_md1118}{}\doxysubsubsubsection{\texorpdfstring{Diffuse Ramp}{Diffuse Ramp}}\label{README.md_autotoc_md1118}
\doxylink{bench__gemm_8cpp_addc86e8508f14411ec98f521c520f875}{A} ramp texture is optionally supported for toon shading effects.\hypertarget{README.md_autotoc_md1119}{}\doxysubsubsubsection{\texorpdfstring{Shadows}{Shadows}}\label{README.md_autotoc_md1119}
Shadows are supported using alpha clipping.\hypertarget{README.md_autotoc_md1120}{}\doxysubsubsubsection{\texorpdfstring{Gradient based Ambient lighting}{Gradient based Ambient lighting}}\label{README.md_autotoc_md1120}
Both lighting modes support using a gradient for ambient light. In \doxylink{struct_vertex}{Vertex} Lit mode the Spherical Harmonics is approximated from the ground, equator and sky colors.\hypertarget{README.md_autotoc_md1121}{}\doxysubsubsubsection{\texorpdfstring{Emission Map}{Emission Map}}\label{README.md_autotoc_md1121}
An optional emission map is supported.\hypertarget{README.md_autotoc_md1122}{}\doxysubsubsubsection{\texorpdfstring{Camera Space Normals}{Camera Space Normals}}\label{README.md_autotoc_md1122}
As sprites are 2d their normals will always be constant. The shaders allow you to define a fixed normal in camera space rather than pass through mesh normals. This not only saves vertex throughput but means lighting looks less \textquotesingle{}flat\textquotesingle{} for rendering sprites with a perspective camera.\hypertarget{README.md_autotoc_md1123}{}\doxysubsubsubsection{\texorpdfstring{Color Adjustment}{Color Adjustment}}\label{README.md_autotoc_md1123}
The shaders allow optional adjustment of hue / saturation and brightness as well as applying a solid color overlay effect for flashing a sprite to a solid color (eg. for damage effects).\hypertarget{README.md_autotoc_md1124}{}\doxysubsubsubsection{\texorpdfstring{Fog}{Fog}}\label{README.md_autotoc_md1124}
Fog is optionally supported\hypertarget{README.md_autotoc_md1125}{}\doxysubsubsection{\texorpdfstring{To Use}{To Use}}\label{README.md_autotoc_md1125}
On your object\textquotesingle{}s material click the drop down for shader and select \doxylink{namespace_spine}{Spine}\textbackslash{}\+Sprite\textbackslash{}\+Pixel Lit, \doxylink{struct_vertex}{Vertex} Lit or Unlit.
