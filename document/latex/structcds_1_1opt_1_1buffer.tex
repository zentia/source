\mbox{[}type-\/option\mbox{]} Option setter for user-\/provided plain buffer 

This option is used by some container as a random access array for storing container\textquotesingle{}s item; for example, a bounded queue may use this option to define underlying buffer implementation.

The template parameter {\ttfamily \doxylink{ir__common_8h_a1ac75f8fee198af783521627f1fecfbc}{Type}} should be rebindable.

Uninitialized buffer is just an array of uninitialized elements. Each element should be manually constructed, for example with a placement new operator. When the uninitialized buffer is destroyed the destructor of its element is not called.

Initialized buffer contains default-\/constructed elements. \doxylink{test__parallel__while_8cpp_a17f27895c682e0ef36fa430ec3c815df}{Element} destructor is called automatically when the buffer is destroyed.

\begin{DoxyNote}{注解}
Usually, initialized and uninitialized buffers are not interchangeable. 
\end{DoxyNote}
