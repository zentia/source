\subsubsection*{template$<$typename \mbox{\hyperlink{_tutorial__sparse__example_8cpp_af639fb96631ba0018f0b2259fc95edfb}{T}}$>$\newline
class tf\+::\+Unbounded\+Task\+Queue$<$ T $>$}
class to create a lock-\/free unbounded work-\/stealing queue 


\begin{DoxyTemplParams}{模板参数}
{\em T} & data type (must be a pointer type)\\
\hline
\end{DoxyTemplParams}


This class implements the work-\/stealing queue described in the paper, \href{https://www.di.ens.fr/~zappa/readings/ppopp13.pdf}{\texttt{ Correct and Efficient Work-\/\+Stealing for Weak Memory Models}}.

Only the queue owner can perform pop and push operations, while others can steal data from the queue simultaneously.
