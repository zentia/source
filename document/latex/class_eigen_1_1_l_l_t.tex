\subsubsection*{template$<$typename \+\_\+\+Matrix\+Type, \mbox{\hyperlink{sugar_8h_ae4f82344f573c70aa1066a2394bba345}{int}} \+\_\+\+Up\+Lo$>$\newline
class Eigen\+::\+LLT$<$ \+\_\+\+Matrix\+Type, \+\_\+\+Up\+Lo $>$}
Standard Cholesky decomposition (LL\texorpdfstring{$^\wedge$}{\string^}T) of a matrix and associated features 


\begin{DoxyTemplParams}{模板参数}
{\em \+\_\+\+Matrix\+Type} & the type of the matrix of which we are computing the LL\texorpdfstring{$^\wedge$}{\string^}T Cholesky decomposition \\
\hline
{\em \+\_\+\+Up\+Lo} & the triangular part that will be used for the decompositon\+: Lower (default) or Upper. The other triangular part won\textquotesingle{}t be read.\\
\hline
\end{DoxyTemplParams}
This class performs a LL\texorpdfstring{$^\wedge$}{\string^}T Cholesky decomposition of a symmetric, positive definite matrix \doxylink{bench__gemm_8cpp_addc86e8508f14411ec98f521c520f875}{A} such that \doxylink{bench__gemm_8cpp_addc86e8508f14411ec98f521c520f875}{A} = LL\texorpdfstring{$^\wedge$}{\string^}\texorpdfstring{$\ast$}{*} = U\texorpdfstring{$^\wedge$}{\string^}\texorpdfstring{$\ast$}{*}U, where L is lower triangular.

While the Cholesky decomposition is particularly useful to solve selfadjoint problems like D\texorpdfstring{$^\wedge$}{\string^}\texorpdfstring{$\ast$}{*}D x = b, for that purpose, we recommend the Cholesky decomposition without square root which is more stable and even faster. Nevertheless, this standard Cholesky decomposition remains useful in many other situations like generalised eigen problems with hermitian matrices.

Remember that Cholesky decompositions are not rank-\/revealing. This \doxylink{class_eigen_1_1_l_l_t}{LLT} decomposition is only stable on positive definite matrices, use \doxylink{class_eigen_1_1_l_d_l_t}{LDLT} instead for the semidefinite case. Also, do not use a Cholesky decomposition to determine whether a system of equations has a solution.

{\bfseries{Performance\+:}} for best performance, it is recommended to use a column-\/major storage format with the Lower triangular part (the default), or, equivalently, a row-\/major storage format with the Upper triangular part. Otherwise, you might get a 20\% slowdown for the full factorization step, and rank-\/updates can be up to 3 times slower.

This class supports the \doxylink{group___inplace_decomposition}{inplace decomposition } mechanism.

Note that during the decomposition, only the lower (or upper, as defined by \+\_\+\+Up\+Lo) triangular part of \doxylink{bench__gemm_8cpp_addc86e8508f14411ec98f521c520f875}{A} is considered. Therefore, the strict lower part does not have to store correct values.
