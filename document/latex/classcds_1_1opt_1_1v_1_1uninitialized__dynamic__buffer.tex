Dynamically allocated uninitialized buffer 

One of available type for {\ttfamily \doxylink{structcds_1_1opt_1_1buffer}{opt\+::buffer}} option.

This buffer maintains dynamically allocated array of uninitialized elements. You should manually construct each element when needed. Allocation is performed at construction time.

\begin{DoxyParagraph}{Template parameters\+:}

\begin{DoxyItemize}
\item {\ttfamily T} -\/ item type storing in the buffer
\item {\ttfamily Alloc} -\/ an allocator used for allocating internal buffer ({\ttfamily std\+::allocator} interface)
\item {\ttfamily Exp2} -\/ a boolean flag. If it is {\ttfamily true} the buffer capacity must be power of two. Otherwise it can be any positive number. Usually, it is required that the buffer has size of a power of two. 
\end{DoxyItemize}
\end{DoxyParagraph}
