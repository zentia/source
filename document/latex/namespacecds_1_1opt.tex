\doxysubsection{详细描述}
Framework to define template options 

There are two kind of options\+:
\begin{DoxyItemize}
\item {\ttfamily type-\/option} -\/ option that determines a data type. The template argument {\ttfamily \doxylink{ir__common_8h_a1ac75f8fee198af783521627f1fecfbc}{Type}} of the option is a type.
\item {\ttfamily value-\/option} -\/ option that determines a value. The template argument {\ttfamily \doxylink{struct_value}{Value}} of the option is a value. 
\end{DoxyItemize}
