Metafunction converting option list to \doxylink{classcds_1_1container_1_1_bronson_a_v_l_tree_map}{Bronson\+AVLTree\+Map} traits 

Note that there are two main specialization of Bronson et al AVL-\/tree\+:
\begin{DoxyItemize}
\item \doxylink{classcds_1_1container_1_1_bronson_a_v_l_tree_map_3_01cds_1_1urcu_1_1gc_3_01_r_c_u_01_4_00_01_key_00_01_t_01_5_00_01_traits_01_4_cds_container_BronsonAVLTreeMap_rcu_ptr}{pointer-\/oriented} -\/ the tree node stores an user-\/provided pointer to value
\item \doxylink{classcds_1_1container_1_1_bronson_a_v_l_tree_map_3_01cds_1_1urcu_1_1gc_3_01_r_c_u_01_4_00_01_key_00_01_t_00_01_traits_01_4_cds_container_BronsonAVLTreeMap_rcu}{data-\/oriented} -\/ the tree node contains a copy of values
\end{DoxyItemize}

\doxylink{struct_depends}{Depends} on tree specialization, different options can be specified.

{\ttfamily Options} are\+:
\begin{DoxyItemize}
\item {\ttfamily \doxylink{structcds_1_1opt_1_1compare}{opt\+::compare}} -\/ key compare functor. No default functor is provided. If the option is not specified, {\ttfamily opt\+::less} is used.
\item {\ttfamily \doxylink{structcds_1_1opt_1_1less}{opt\+::less}} -\/ specifies binary predicate used for key compare. At least {\ttfamily opt\+::compare} or {\ttfamily opt\+::less} should be defined.
\item {\ttfamily \doxylink{structcds_1_1opt_1_1node__allocator}{opt\+::node\+\_\+allocator}} -\/ the allocator for internal nodes. Default is \doxylink{external_2libcds_2cds_2user__setup_2allocator_8h_ad5fb7e3cbe8cf24295f0ea3c165a2d82}{CDS\+\_\+\+DEFAULT\+\_\+\+ALLOCATOR}.
\item {\ttfamily \doxylink{structcds_1_1opt_1_1allocator}{opt\+::allocator}} -\/ the allocator for node\textquotesingle{}s value. Default is \doxylink{external_2libcds_2cds_2user__setup_2allocator_8h_ad5fb7e3cbe8cf24295f0ea3c165a2d82}{CDS\+\_\+\+DEFAULT\+\_\+\+ALLOCATOR}. This option is not used in {\ttfamily \doxylink{classcds_1_1container_1_1_bronson_a_v_l_tree_map}{Bronson\+AVLTree\+Map$<$\+RCU, Key, T\texorpdfstring{$\ast$}{*}, Traits$>$}} specialisation
\item {\ttfamily \doxylink{structcds_1_1intrusive_1_1opt_1_1disposer}{cds\+::intrusive\+::opt\+::disposer}} -\/ the functor used for dispose removed values. The user-\/provided disposer is used only for pointer-\/oriented tree specialization like {\ttfamily \doxylink{classcds_1_1container_1_1_bronson_a_v_l_tree_map}{Bronson\+AVLTree\+Map$<$\+GC, Key, T\texorpdfstring{$\ast$}{*}, Traits$>$}}. When the node becomes the rounting node without value, the disposer will be called to signal that the memory for the value can be safely freed. Default is {\ttfamily cds\+::intrusive\+::opt\+::delete\+\_\+disposer} which calls {\ttfamily delete} operator. Due the nature of GC schema the disposer may be called asynchronously.
\item {\ttfamily \doxylink{structcds_1_1opt_1_1sync__monitor}{opt\+::sync\+\_\+monitor}} -\/ \doxysectlink{cds_sync_monitor}{synchronization monitor}{0} type for node-\/level locking, default is {\ttfamily \doxylink{classcds_1_1sync_1_1injecting__monitor}{cds\+::sync\+::injecting\+\_\+monitor$<$cds\+::sync\+::spin$>$}} 
\item {\ttfamily \doxylink{structcds_1_1container_1_1bronson__avltree_1_1relaxed__insert}{bronson\+\_\+avltree\+::relaxed\+\_\+insert}} -\/ enable ({\ttfamily true}) or disable ({\ttfamily false}, the default) \doxylink{structcds_1_1container_1_1bronson__avltree_1_1relaxed__insert}{relaxed insertion}
\item {\ttfamily \doxylink{structcds_1_1opt_1_1item__counter}{opt\+::item\+\_\+counter}} -\/ the type of item counting feature, by default it is disabled ({\ttfamily \doxylink{classcds_1_1atomicity_1_1empty__item__counter}{atomicity\+::empty\+\_\+item\+\_\+counter}}) To enable it use {\ttfamily \doxylink{classcds_1_1atomicity_1_1item__counter}{atomicity\+::item\+\_\+counter}} or {\ttfamily \doxylink{classcds_1_1atomicity_1_1cache__friendly__item__counter}{atomicity\+::cache\+\_\+friendly\+\_\+item\+\_\+counter}} 
\item {\ttfamily \doxylink{structcds_1_1opt_1_1memory__model}{opt\+::memory\+\_\+model}} -\/ C++ memory ordering model. Can be {\ttfamily \doxylink{structcds_1_1opt_1_1v_1_1relaxed__ordering}{opt\+::v\+::relaxed\+\_\+ordering}} (relaxed memory model, the default) or {\ttfamily \doxylink{structcds_1_1opt_1_1v_1_1sequential__consistent}{opt\+::v\+::sequential\+\_\+consistent}} (sequentially consisnent memory model).
\item {\ttfamily \doxylink{structcds_1_1opt_1_1stat}{opt\+::stat}} -\/ internal statistics, by default it is disabled ({\ttfamily \doxylink{structcds_1_1container_1_1bronson__avltree_1_1empty__stat}{bronson\+\_\+avltree\+::empty\+\_\+stat}}) To enable statistics use {\ttfamily {\ttfamily \doxylink{structcds_1_1container_1_1bronson__avltree_1_1stat}{bronson\+\_\+avltree\+::stat}} {\ttfamily opt\+::backoff} -\/} back-\/off strategy, by default no strategy is used ({\ttfamily \doxylink{structcds_1_1backoff_1_1empty}{cds\+::backoff\+::empty}})
