Jacobi preconditioner for \doxylink{class_eigen_1_1_least_squares_conjugate_gradient}{Least\+Squares\+Conjugate\+Gradient} 

This class allows to approximately solve for \doxylink{bench__gemm_8cpp_addc86e8508f14411ec98f521c520f875}{A}\textquotesingle{} \doxylink{bench__gemm_8cpp_addc86e8508f14411ec98f521c520f875}{A} x = \doxylink{bench__gemm_8cpp_addc86e8508f14411ec98f521c520f875}{A}\textquotesingle{} b problems assuming \doxylink{bench__gemm_8cpp_addc86e8508f14411ec98f521c520f875}{A}\textquotesingle{} \doxylink{bench__gemm_8cpp_addc86e8508f14411ec98f521c520f875}{A} is a diagonal matrix. In other words, this preconditioner neglects all off diagonal entries and, in \doxylink{namespace_eigen}{Eigen}\textquotesingle{}s language, solves for\+:
