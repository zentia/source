class Minimal$<$ T $>$}
\doxylink{struct_wrapper}{Wrapper} around T where all members are private. 

Used to prove that aligned\+\_\+space$<$\+T,\+N$>$ never calls member of T.

Has tightly controlled interface so that we can verify that parallel\+\_\+sort uses only the required interface.
