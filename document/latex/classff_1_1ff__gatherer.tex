\doxysubsection{详细描述}
\doxylink{bench__gemm_8cpp_addc86e8508f14411ec98f521c520f875}{A} class representing the {\itshape Collector} node in a {\itshape Farm} skeleton. 

This class models the {\ttfamily gatherer}, which wraps all the methods and structures used by the {\itshape Collector} node in a {\ttfamily Farm} skeleton. The {\ttfamily farm} can be seen as a three-\/stages {\ttfamily pipeline}, the stages being a {\ttfamily \doxylink{classff_1_1ff__loadbalancer}{ff\+\_\+loadbalancer}} called {\itshape emitter}, a pool of {\ttfamily \doxylink{classff_1_1ff__node}{ff\+\_\+node}} called {\itshape workers} and -\/ optionally -\/ a {\ttfamily \doxylink{classff_1_1ff__gatherer}{ff\+\_\+gatherer}} called {\itshape collector}. The {\itshape Collector} node can be used to gather the results coming from the computations executed by the pool of {\itshape workers}. The {\itshape collector} can also be connected to the {\itshape emitter} node via a feedback channel, in order to create a {\ttfamily farm-\/with-\/feedback} skeleton.
