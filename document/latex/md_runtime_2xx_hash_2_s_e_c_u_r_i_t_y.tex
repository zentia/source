Security updates are applied only to the latest release.\hypertarget{md_runtime_2xx_hash_2_s_e_c_u_r_i_t_y_autotoc_md457}{}\doxysection{\texorpdfstring{Reporting a Vulnerability}{Reporting a Vulnerability}}\label{md_runtime_2xx_hash_2_s_e_c_u_r_i_t_y_autotoc_md457}
If you have discovered a security vulnerability in this project, please report it privately. {\bfseries{Do not disclose it as a public issue.}} This gives us time to work with you to fix the issue before public exposure, reducing the chance that the exploit will be used before a patch is released.

Please disclose it at \href{https://github.com/Cyan4973/xxHash/security/advisories/new}{\texttt{ security advisory}}.

This project is maintained by a team of volunteers on a reasonable-\/effort basis. As such, please give us at least 90 days to work on a fix before public exposure.
