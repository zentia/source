class cds\+::intrusive\+::\+Cached\+Free\+List$<$ Free\+List, Cache\+Size, Padding $>$}
Cached free list 

The class that is a wrapper over other {\ttfamily \doxylink{classcds_1_1intrusive_1_1_free_list}{Free\+List}} contains a small cache of free elements. Before placing a new item into underlying {\ttfamily \doxylink{classcds_1_1intrusive_1_1_free_list}{Free\+List}} the cached free-\/list tryes to put that item into the cache if its corresponding slot is empty. The slot is calculated by current thread id\+: 
\begin{DoxyCode}{0}
\DoxyCodeLine{\textcolor{keywordtype}{int}\ slot\ =\ std::hash<std::thread::id>()(\ std::this\_thread::get\_id())\ \&\ (CacheSize\ -\/\ 1);}

\end{DoxyCode}


When getting the free-\/list checks the corresponding cache slot. If it is not empty, its contents is returned.

In some cases such simple algorithm significantly reduces {\ttfamily \doxylink{classcds_1_1intrusive_1_1_free_list}{Free\+List}} contention.

Template parameters\+:
\begin{DoxyItemize}
\item {\ttfamily \doxylink{classcds_1_1intrusive_1_1_free_list}{Free\+List}} -\/ a free-\/list implementation\+: {\ttfamily \doxylink{classcds_1_1intrusive_1_1_free_list}{Free\+List}}, {\ttfamily \doxylink{classcds_1_1intrusive_1_1_tagged_free_list}{Tagged\+Free\+List}} 
\item {\ttfamily Cache\+Size} -\/ size of cache, a small power-\/of-\/two number, default is 16
\item {\ttfamily Padding} -\/ padding of cache elements for solving false sharing, default is {\ttfamily cds\+::c\+\_\+n\+Cache\+Line\+Size} 
\end{DoxyItemize}
