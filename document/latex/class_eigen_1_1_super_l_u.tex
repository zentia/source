\subsubsection*{template$<$typename \+\_\+\+Matrix\+Type$>$\newline
class Eigen\+::\+Super\+LU$<$ \+\_\+\+Matrix\+Type $>$}
\doxylink{bench__gemm_8cpp_addc86e8508f14411ec98f521c520f875}{A} sparse direct LU factorization and solver based on the \doxylink{class_eigen_1_1_super_l_u}{Super\+LU} library 

This class allows to solve for A.\+X = \doxylink{bench__gemm_8cpp_a37a83060ac796961b44991c836f083f7}{B} sparse linear problems via a direct LU factorization using the \doxylink{class_eigen_1_1_super_l_u}{Super\+LU} library. The sparse matrix \doxylink{bench__gemm_8cpp_addc86e8508f14411ec98f521c520f875}{A} must be squared and invertible. The vectors or matrices X and \doxylink{bench__gemm_8cpp_a37a83060ac796961b44991c836f083f7}{B} can be either dense or sparse.
