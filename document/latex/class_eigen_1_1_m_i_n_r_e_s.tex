\subsubsection*{template$<$typename \+\_\+\+Matrix\+Type, \mbox{\hyperlink{sugar_8h_ae4f82344f573c70aa1066a2394bba345}{int}} \+\_\+\+Up\+Lo, typename \+\_\+\+Preconditioner$>$\newline
class Eigen\+::\+MINRES$<$ \+\_\+\+Matrix\+Type, \+\_\+\+Up\+Lo, \+\_\+\+Preconditioner $>$}
\doxylink{bench__gemm_8cpp_addc86e8508f14411ec98f521c520f875}{A} minimal residual solver for sparse symmetric problems 

This class allows to solve for A.\+x = b sparse linear problems using the \doxylink{class_eigen_1_1_m_i_n_r_e_s}{MINRES} algorithm of Paige and Saunders (1975). The sparse matrix \doxylink{bench__gemm_8cpp_addc86e8508f14411ec98f521c520f875}{A} must be symmetric (possibly indefinite). The vectors x and b can be either dense or sparse.


\begin{DoxyTemplParams}{模板参数}
{\em \+\_\+\+Matrix\+Type} & the type of the sparse matrix \doxylink{bench__gemm_8cpp_addc86e8508f14411ec98f521c520f875}{A}, can be a dense or a sparse matrix. \\
\hline
{\em \+\_\+\+Up\+Lo} & the triangular part that will be used for the computations. It can be Lower, Upper, or Lower\texorpdfstring{$\vert$}{|}\+Upper in which the full matrix entries will be considered. Default is Lower. \\
\hline
{\em \+\_\+\+Preconditioner} & the type of the preconditioner. Default is \doxylink{class_eigen_1_1_diagonal_preconditioner}{Diagonal\+Preconditioner}\\
\hline
\end{DoxyTemplParams}
The maximal number of iterations and tolerance value can be controlled via the \doxylink{class_eigen_1_1_iterative_solver_base_a1bbaf86b7df6d54e78ced4da6ed2dac1}{set\+Max\+Iterations()} and \doxylink{class_eigen_1_1_iterative_solver_base_ac43c50c69a5b621cb5a9991a1005750d}{set\+Tolerance()} methods. The defaults are the size of the problem for the maximal number of iterations and \doxylink{struct_eigen_1_1_generic_num_traits_aadcf7fa75b2bc9a0e461a3b851d836a7}{Num\+Traits$<$\+Scalar$>$\+::epsilon()} for the tolerance.
