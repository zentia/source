Forward iterator 

The forward iterator for Michael\textquotesingle{}s set is based on {\ttfamily Ordered\+List} forward iterator and has some features\+:
\begin{DoxyItemize}
\item it has no post-\/increment operator
\item it iterates items in unordered fashion
\item The iterator cannot be moved across thread boundary because it may contain GC\textquotesingle{}s guard that is thread-\/private GC data.
\end{DoxyItemize}

Iterator thread safety depends on type of {\ttfamily Ordered\+List\+:} 
\begin{DoxyItemize}
