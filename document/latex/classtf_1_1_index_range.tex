class tf\+::\+Index\+Range$<$ T $>$}
class to create an index range of integral indices with a step size 

This class provides functionality for managing a range of indices, where the range is defined by a starting index, an ending index, and a step size. The indices must be of an integral type. For example, the range \mbox{[}0, 10) with a step size 2 represents the five elements, 0, 2, 4, 6, and 8.

\begin{DoxyReturn}{返回}
a new \doxylink{classtf_1_1_index_range}{Index\+Range} object representing the given discrete domain
\end{DoxyReturn}
The discrete domain of a range refers to a counter-\/based sequence indexed from 0 to {\ttfamily N}, where {\ttfamily N} is the size (i.\+e., number of iterated elements) of the range. For example, a discrete domain of the range \mbox{[}0, 10) with a step size of 2 corresponds to the sequence 0, 1, 2, 3, and 4, which map to the range elements 0, 2, 4, 6, and 8.

For a partitioned domain \mbox{[}{\ttfamily part\+\_\+beg}, {\ttfamily part\+\_\+end}), this function returns the corresponding range. For instance, the partitioned domain \mbox{[}2, 5) for the above example returns the range \mbox{[}4, 10) with the same step size of 2.

\begin{DoxyAttention}{注意}
Users must ensure the specified domain is valid with respect to the range. 
\end{DoxyAttention}
