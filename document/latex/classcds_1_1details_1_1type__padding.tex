Automatic alignment type {\ttfamily T} to {\ttfamily Align\+Factor} 

The class adds appropriate bytes to type T that the following condition is true\+: 
\begin{DoxyCode}{0}
\DoxyCodeLine{\textcolor{keyword}{sizeof}(\ \mbox{\hyperlink{classcds_1_1details_1_1type__padding_ab342c077cac052ca794814cf6d57607d}{type\_padding<T,AlignFactor>::type}}\ )\ \%\ AlignFactor\ ==\ 0}

\end{DoxyCode}
 It is guaranteed that count of padding bytes no more than Align\+Factor -\/ 1.

{\bfseries{Applicability\+:}} type {\ttfamily T} must not have constructors another that default ctor. For example, {\ttfamily T} may be any POD type.
