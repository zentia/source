\doxysubsection{详细描述}
Converts a human-\/readable size string (with unit literal) to uin64\+\_\+t size. Example\+: "{}100"{} =\texorpdfstring{$>$}{>} 100 "{}1 b"{} =\texorpdfstring{$>$}{>} 100 "{}10\+Kb"{} =\texorpdfstring{$>$}{>} 10240 // you can configure this to be interpreted as kilobyte (\texorpdfstring{$\ast$}{*}1000) or kibibyte (\texorpdfstring{$\ast$}{*}1024) "{}10 KB"{} =\texorpdfstring{$>$}{>} 10240 "{}10 kb"{} =\texorpdfstring{$>$}{>} 10240 "{}10 kib"{} =\texorpdfstring{$>$}{>} 10240 // \texorpdfstring{$\ast$}{*}i, \texorpdfstring{$\ast$}{*}ib are always interpreted as \texorpdfstring{$\ast$}{*}bibyte (\texorpdfstring{$\ast$}{*}1024) "{}10kb"{} =\texorpdfstring{$>$}{>} 10240 "{}2 MB"{} =\texorpdfstring{$>$}{>} 2097152 "{}2 Ei\+B"{} =\texorpdfstring{$>$}{>} 2\texorpdfstring{$^\wedge$}{\string^}61 // Units up to exibyte are supported
