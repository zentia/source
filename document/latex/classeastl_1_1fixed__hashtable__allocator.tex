\doxylink{classeastl_1_1fixed__hashtable__allocator}{fixed\+\_\+hashtable\+\_\+allocator}

Provides a base class for fixed hashtable allocations. To consider\+: Have this inherit from \doxylink{classeastl_1_1fixed__node__allocator}{fixed\+\_\+node\+\_\+allocator}.

Template parameters\+: bucket\+Count The fixed number of hashtable buckets to provide. node\+Count The number of objects the pool contains. node\+Alignment The alignment of the objects to allocate. node\+Alignment\+Offset The alignment offset of the objects to allocate. b\+Enable\+Overflow Whether or not we should use the overflow heap if our object pool is exhausted. Overflow\+Allocator Overflow allocator, which is only used if b\+Enable\+Overflow == true. Defaults to the global heap.
