The function returns the interface version of the \doxylink{struct_t_b_b}{TBB} shared library being used. 

The version it returns is determined at runtime, not at compile/link time. So it can be different than the value of TBB\+\_\+\+INTERFACE\+\_\+\+VERSION obtained at compile time.
