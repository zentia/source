class cds\+::\+OS\+::aligned\+\_\+allocator$<$ T $>$}
Aligned allocator 

This allocator is intended for allocating of an aligned memory block. It uses wrappers around platform-\/specific function for allocating and deallocating the block of memory\+: \begin{DoxyItemize}
\item {\ttfamily aligned\+\_\+malloc} for allocating \item {\ttfamily aligned\+\_\+free} for deallocating\end{DoxyItemize}
The {\ttfamily aligned\+\_\+malloc} function wraps\+: \begin{DoxyItemize}
\item {\ttfamily for} Win\+: {\ttfamily \+\_\+aligned\+\_\+malloc} function \item {\ttfamily for} other OSes\+: {\ttfamily posix\+\_\+memalign} / {\ttfamily memalign} function\end{DoxyItemize}
The {\ttfamily aligned\+\_\+free} function wraps\+: \begin{DoxyItemize}
\item {\ttfamily for} Win\+: {\ttfamily \+\_\+aligned\+\_\+free} function \item {\ttfamily for} other OSes\+: {\ttfamily free} function\end{DoxyItemize}
This class should not be used directly. Use \doxylink{classcds_1_1details_1_1_aligned_allocator}{cds\+::details\+::\+Aligned\+Allocator} instead.
