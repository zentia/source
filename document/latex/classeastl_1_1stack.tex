class eastl\+::stack$<$ T, Container $>$}
stack

stack is an adapter class provides a LIFO (last-\/in, first-\/out) interface via wrapping a sequence that provides at least the following operations\+: push\+\_\+back pop\+\_\+back back

In practice this means vector, deque, string, list, \doxylink{classeastl_1_1intrusive__list}{intrusive\+\_\+list}.
