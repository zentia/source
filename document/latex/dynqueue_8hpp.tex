\doxysubsection{详细描述}
Implementation of a dynamic queue. Not currently used. 

Dynamic (list-\/based) Single-\/\+Writer Single-\/\+Reader (or Single-\/\+Producer Single-\/\+Consumer) unbounded queue.

No lock is needed around pop and push methods. See also \doxylink{ubuffer_8hpp}{ubuffer.\+hpp} for a more efficient SPSC unbounded queue.

M. Aldinucci, M. Danelutto, P. Kilpatrick, M. Meneghin, and M. Torquati, "{}\+An Efficient Unbounded Lock-\/\+Free Queue for Multi-\/core Systems,"{} in Proc. of 18th Intl. Euro-\/\+Par 2012 Parallel Processing, Rhodes Island, Greece, 2012, pp. 662-\/673. doi\+:10.\+1007/978-\/3-\/642-\/32820-\/6\+\_\+65
