\doxysubsubsubsection{详细描述}
Markers represent a single discreet event in time. Markers have a scope, described by an enumerated type \doxylink{group__markers_ga9712c6a992a435d4d134e7164f609f04}{\+\_\+\+\_\+itt\+\_\+scope}. Markers are created by the API call \+\_\+\+\_\+itt\+\_\+marker. \doxylink{bench__gemm_8cpp_addc86e8508f14411ec98f521c520f875}{A} marker instance can be given an ID for use in adding metadata.
