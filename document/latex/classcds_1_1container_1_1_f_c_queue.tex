Flat-\/combining queue 

\doxylink{namespacecds_1_1algo_1_1flat__combining_cds_flat_combining_description}{Flat combining} sequential queue. The class can be considered as a concurrent FC-\/based wrapper for {\ttfamily std\+::queue}.

Template parameters\+:
\begin{DoxyItemize}
\item {\ttfamily T} -\/ a value type stored in the queue
\item {\ttfamily Queue} -\/ sequential queue implementation, default is {\ttfamily std\+::queue$<$\+T$>$} 
\item {\ttfamily Trats} -\/ type traits of flat combining, default is {\ttfamily \doxylink{structcds_1_1container_1_1fcqueue_1_1traits}{fcqueue\+::traits}}. {\ttfamily \doxylink{structcds_1_1container_1_1fcqueue_1_1make__traits}{fcqueue\+::make\+\_\+traits}} metafunction can be used to construct {\ttfamily fcqueue\+::traits} specialization. 
\end{DoxyItemize}
