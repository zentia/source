\doxylink{bench__gemm_8cpp_addc86e8508f14411ec98f521c520f875}{A} high-\/performance thread-\/safe blocking concurrent bounded queue. 

This is the pre-\/\+PPL \doxylink{struct_t_b_b}{TBB} concurrent queue which supports boundedness and blocking semantics. Note that method names agree with the PPL-\/style concurrent queue. Multiple threads may each push and pop concurrently. Assignment construction is not allowed.
