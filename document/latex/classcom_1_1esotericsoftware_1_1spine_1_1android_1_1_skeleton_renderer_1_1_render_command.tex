\doxysubsection{详细描述}
Stores the vertices, indices, and atlas page index to be used for rendering one or more attachments of a {\ttfamily \doxylink{classcom_1_1esotericsoftware_1_1spine_1_1_skeleton}{Skeleton}} to a {\ttfamily \doxylink{}{Canvas}}. See the implementation of {\ttfamily \doxylink{classcom_1_1esotericsoftware_1_1spine_1_1android_1_1_skeleton_renderer_ad2d7bf62da1d2cf610e79b598b10f6e0}{Skeleton\+Renderer\#render(\+Skeleton)}} and {\ttfamily \doxylink{}{Skeleton\+Renderer\#render\+To\+Canvas(\+Canvas, Array)}} on how to use this data to render it to a {\ttfamily \doxylink{}{Canvas}}.
