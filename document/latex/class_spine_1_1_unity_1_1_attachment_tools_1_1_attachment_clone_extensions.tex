When parameter {\ttfamily premultiply\+Alpha} is set to {\ttfamily true}, a premultiply alpha clone of the original texture will be created. Additionally, this PMA Texture clone is cached for later re-\/use, which might steadily increase the Texture memory footprint when used excessively.
