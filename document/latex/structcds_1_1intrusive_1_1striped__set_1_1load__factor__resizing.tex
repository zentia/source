Load factor based resizing policy 

When total item count in a container exceeds {\ttfamily container.\+bucket\+\_\+count() \texorpdfstring{$\ast$}{*} Load\+Factor} then resizing is needed.

This policy is stateless.

The {\ttfamily \doxylink{structcds_1_1intrusive_1_1striped__set_1_1load__factor__resizing_a60d29e5332c5fd63ee42dfb69cad873f}{reset()}} function is called after the resizing is done. The function is intended for resetting internal state of the policy.
