An \doxylink{structeastl_1_1intrusive__hashtable__iterator__base}{intrusive\+\_\+hashtable\+\_\+iterator\+\_\+base} iterates the entire hash table and not just nodes within a single bucket. Users in general will use a hash table iterator much more often, as it is much like other container iterators (e.\+g. \doxylink{classeastl_1_1vector_a4bae2bde2429116a3cb0e31b2168ccbe}{vector\+::iterator}).

We define a base class here because it is shared by both const and non-\/const iterators.
