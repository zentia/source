The kind of relation between two instances is specified by the enumerated type \doxylink{group__relations_ga2f588e0d778ac0cca7cc0b64cfb1bdca}{\+\_\+\+\_\+itt\+\_\+relation}. Relations between instances can be added with an API call. The relation API uses instance IDs.
