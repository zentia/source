\doxylink{bench__gemm_8cpp_addc86e8508f14411ec98f521c520f875}{A} conjugate gradient solver for sparse (or dense) least-\/square problems 

This class allows to solve for \doxylink{bench__gemm_8cpp_addc86e8508f14411ec98f521c520f875}{A} x = b linear problems using an iterative conjugate gradient algorithm. The matrix \doxylink{bench__gemm_8cpp_addc86e8508f14411ec98f521c520f875}{A} can be non symmetric and rectangular, but the matrix \doxylink{bench__gemm_8cpp_addc86e8508f14411ec98f521c520f875}{A}\textquotesingle{} \doxylink{bench__gemm_8cpp_addc86e8508f14411ec98f521c520f875}{A} should be positive-\/definite to guaranty stability. Otherwise, the \doxylink{class_eigen_1_1_sparse_l_u}{Sparse\+LU} or \doxylink{class_eigen_1_1_sparse_q_r}{Sparse\+QR} classes might be preferable. The matrix \doxylink{bench__gemm_8cpp_addc86e8508f14411ec98f521c520f875}{A} and the vectors x and b can be either dense or sparse.


\begin{DoxyTemplParams}{模板参数}
{\em \+\_\+\+Matrix\+Type} & the type of the matrix \doxylink{bench__gemm_8cpp_addc86e8508f14411ec98f521c520f875}{A}, can be a dense or a sparse matrix. \\
\hline
{\em \+\_\+\+Preconditioner} & the type of the preconditioner. Default is \doxylink{class_eigen_1_1_least_square_diagonal_preconditioner}{Least\+Square\+Diagonal\+Preconditioner}\\
\hline
\end{DoxyTemplParams}
\textbackslash{}implsparsesolverconcept

The maximal number of iterations and tolerance value can be controlled via the \doxylink{class_eigen_1_1_iterative_solver_base_a1bbaf86b7df6d54e78ced4da6ed2dac1}{set\+Max\+Iterations()} and \doxylink{class_eigen_1_1_iterative_solver_base_ac43c50c69a5b621cb5a9991a1005750d}{set\+Tolerance()} methods. The defaults are the size of the problem for the maximal number of iterations and \doxylink{struct_eigen_1_1_generic_num_traits_aadcf7fa75b2bc9a0e461a3b851d836a7}{Num\+Traits$<$\+Scalar$>$\+::epsilon()} for the tolerance.
