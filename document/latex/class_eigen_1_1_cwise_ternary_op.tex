Generic expression where a coefficient-\/wise ternary operator is applied to two expressions

\hline
\end{DoxyTemplParams}
This class represents an expression where a coefficient-\/wise ternary operator is applied to three expressions. It is the return type of ternary operators, by which we mean only those ternary operators where all three arguments are \doxylink{namespace_eigen}{Eigen} expressions. For example, the return type of betainc(matrix1, matrix2, matrix3) is a \doxylink{class_eigen_1_1_cwise_ternary_op}{Cwise\+Ternary\+Op}.

Most of the time, this is the only way that it is used, so you typically don\textquotesingle{}t have to name \doxylink{class_eigen_1_1_cwise_ternary_op}{Cwise\+Ternary\+Op} types explicitly.
