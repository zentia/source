\doxysubsubsection{详细描述}
Contains functions used in the classic 32-\/bit xx\+Hash algorithm.

\begin{DoxyNote}{注解}
XXH32 is useful for older platforms, with no or poor 64-\/bit performance. Note that the \doxylink{group___x_x_h3__family}{XXH3 family} provides competitive speed for both 32-\/bit and 64-\/bit systems, and offers true 64/128 bit hash results.
\end{DoxyNote}
