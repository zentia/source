class tbb\+::movable\+\_\+exception$<$ Exception\+Data $>$}
Template that can be used to implement exception that transfers arbitrary Exception\+Data to the root thread 

Code using \doxylink{struct_t_b_b}{TBB} can instantiate this template with an arbitrary Exception\+Data type and throw this exception object. Such exceptions are intercepted by the \doxylink{struct_t_b_b}{TBB} scheduler and delivered to the root thread (). \begin{DoxySeeAlso}{参见}
