\doxysubsection{详细描述}
Stripped down version of spin\+\_\+mutex. 

Instances of \doxylink{class_malloc_mutex}{Malloc\+Mutex} must be declared in memory that is zero-\/initialized. There are no constructors. This is a feature that lets it be used in situations where the mutex might be used while file-\/scope constructors are running.

There are no methods "{}acquire"{} or "{}release"{}. The \doxylink{class_malloc_mutex_1_1scoped__lock}{scoped\+\_\+lock} must be used in a strict block-\/scoped locking pattern. Omitting these methods permitted further simplification.
