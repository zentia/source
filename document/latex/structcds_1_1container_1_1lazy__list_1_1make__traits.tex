{\ttfamily Options} are\+:
\begin{DoxyItemize}

Default {\ttfamily is} \doxylink{namespacecds_1_1sync_a3d645de2cdf4530abac4f1ba0d293ba6}{cds\+::sync\+::spin}. \item {\ttfamily \doxylink{structcds_1_1opt_1_1equal__to}{opt\+::equal\+\_\+to}} -\/ specifies binary functor for comparing keys for equality. This option is applicable only for unordered list. \item {\ttfamily \doxylink{structcds_1_1opt_1_1sort}{opt\+::sort}} -\/ specifies ordering policy. Default value is {\ttfamily true}, i.\+e. the list is ordered. Note\+: unordering feature is not fully supported yet.

\item {\ttfamily \doxylink{structcds_1_1opt_1_1item__counter}{opt\+::item\+\_\+counter}} -\/ the type of item counting feature. Default is disabled ({\ttfamily \doxylink{classcds_1_1atomicity_1_1empty__item__counter}{atomicity\+::empty\+\_\+item\+\_\+counter}}). To enable item counting use {\ttfamily \doxylink{classcds_1_1atomicity_1_1item__counter}{atomicity\+::item\+\_\+counter}} or {\ttfamily \doxylink{classcds_1_1atomicity_1_1cache__friendly__item__counter}{atomicity\+::cache\+\_\+friendly\+\_\+item\+\_\+counter}} 
\item {\ttfamily \doxylink{structcds_1_1opt_1_1stat}{opt\+::stat}} -\/ internal statistics.
