\doxysubsection{详细描述}
An asset loader to create and load skeleton data. The data file is assumed to be binary if it ends with {\ttfamily .skel}, otherwise JSON is assumed. The {\ttfamily \doxylink{classcom_1_1esotericsoftware_1_1spine_1_1utils_1_1_skeleton_data_loader_1_1_skeleton_data_parameter}{Skeleton\+Data\+Parameter}} can provide a texture atlas name or an {\ttfamily \doxylink{interfacecom_1_1esotericsoftware_1_1spine_1_1attachments_1_1_attachment_loader}{Attachment\+Loader}}. If neither is provided, a texture atlas name based on the skeleton file name with an {\ttfamily .atlas} extension is used. When a texture atlas name is used, the texture atlas is loaded by the asset manager as a dependency. 

Example\+:


\begin{DoxyPre}
// Load skeleton.json and skeleton.atlas:
assetManager.load("{}skeleton.json"{}, SkeletonData.class);
// Or specify the atlas/AttachmentLoader and scale:
assetManager.setLoader(SkeletonData.class, new SkeletonDataLoader(new InternalFileHandleResolver()));
\doxylink{classcom_1_1esotericsoftware_1_1spine_1_1utils_1_1_skeleton_data_loader_1_1_skeleton_data_parameter}{SkeletonDataParameter} parameter = new \doxylink{classcom_1_1esotericsoftware_1_1spine_1_1utils_1_1_skeleton_data_loader_1_1_skeleton_data_parameter}{SkeletonDataParameter}("{}skeleton2x.atlas"{}, 2);
assetManager.load("{}skeleton.json"{}, SkeletonData.class, parameter);
\end{DoxyPre}
