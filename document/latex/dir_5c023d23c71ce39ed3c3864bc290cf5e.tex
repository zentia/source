\doxysubsection{详细描述}
The spine-\/godot runtime provides functionality to load, manipulate and render \href{http://esotericsoftware.com}{\texttt{ Spine}} skeletal animation data using \href{https://godotengine.org/}{\texttt{ Godot}} 3.\+5.\+x and 4.\+0.\+x respectively. spine-\/godot is based on \href{../spine-cpp}{\texttt{ spine-\/cpp}}.\hypertarget{README.md_autotoc_md1028}{}\doxysubsection{\texorpdfstring{See the \href{http://esotericsoftware.com/spine-godot}{\texttt{ spine-\/godot documentation}} for in-\/depth information.}{See the \href{http://esotericsoftware.com/spine-godot}{\texttt{ spine-\/godot documentation}} for in-\/depth information.}}\label{README.md_autotoc_md1028}
