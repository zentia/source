\subsubsection*{template$<$typename T, typename Container = eastl\+::deque$<$\+T, EASTLAllocator\+Type, DEQUE\+\_\+\+DEFAULT\+\_\+\+SUBARRAY\+\_\+\+SIZE(\+T)$>$$>$\newline
class eastl\+::queue$<$ T, Container $>$}
queue

queue is an adapter class provides a FIFO (first-\/in, first-\/out) interface via wrapping a sequence that provides at least the following operations\+: push\+\_\+back pop\+\_\+front front back

In practice this usually means deque, list, \doxylink{classeastl_1_1intrusive__list}{intrusive\+\_\+list}. vector and string ~\newline
 cannot be used because they don\textquotesingle{}t provide pop-\/front. This is reasonable because a vector or string pop\+\_\+front would be inefficient and could lead to silently poor performance.
