This release includes several new changes such as CPU-\/\+GPU tasking, algorithm collection, enhanced web-\/based profiler, documentation, and unit tests.\hypertarget{release-3-1-0_release-3-1-0_download}{}\doxysubsection{\texorpdfstring{Download}{Download}}\label{release-3-1-0_release-3-1-0_download}
