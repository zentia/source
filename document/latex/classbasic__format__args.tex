class basic\+\_\+format\+\_\+args$<$ Context $>$}
A view of a collection of formatting arguments. To avoid lifetime issues it should only be used as a parameter type in type-\/erased functions such as {\ttfamily vformat}\+: \begin{DoxyVerb}void vlog(fmt::string_view fmt, fmt::format_args args);  // OK
