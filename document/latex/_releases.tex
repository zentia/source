This page summarizes the release notes of Taskflow. We classify each release with three numbers\+:

{\ttfamily Major.\+Minor.\+Patch}

\doxylink{bench__gemm_8cpp_addc86e8508f14411ec98f521c520f875}{A} {\itshape major} release indicates significant codebase changes and API modifications, a {\itshape minor} release indicates technical improvement over a major release line, and a {\itshape patch} release indicates fixes of bugs and other issues.

All releases are available in @\+Project\+Git\+Hub.
