\doxysubsection{详细描述}
The class handles access to the single instance of market, and to TLS to keep scheduler instances. 

It also supports automatic on-\/demand initialization of the \doxylink{struct_t_b_b}{TBB} scheduler. The class contains only static data members and methods.
