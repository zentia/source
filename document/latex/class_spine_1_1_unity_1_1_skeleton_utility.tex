If true, \doxylink{class_spine_1_1_skeleton_a0f883b19b2bc80d4d5db65fba9f1f99c}{Skeleton.\+ScaleX} and \doxylink{class_spine_1_1_skeleton_a43db6d20181ea0ddbb8710df9d0eba3a}{Skeleton.\+ScaleY} are followed by 180 degree rotation. Therefore, it is recommended to enable this parameter where possible. When creating hinge chains for a chain of skeleton bones via \doxylink{class_spine_1_1_unity_1_1_skeleton_utility_bone}{Skeleton\+Utility\+Bone}, it is mandatory to have {\ttfamily flip\+By180\+Degree\+Rotation} enabled.
