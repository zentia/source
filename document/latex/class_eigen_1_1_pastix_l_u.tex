Interface to the Pa\+Stix solver 

\doxylink{struct_eigen_1_1_sparse}{Sparse} direct LU solver based on Pa\+StiX library

This class is used to solve the linear systems A.\+X = \doxylink{bench__gemm_8cpp_a37a83060ac796961b44991c836f083f7}{B} via the Pa\+Stix library. The matrix can be either real or complex, symmetric or not.

\begin{DoxySeeAlso}{参见}
Tutorial\+Sparse\+Direct\+Solvers
\end{DoxySeeAlso}
This class is used to solve the linear systems A.\+X = \doxylink{bench__gemm_8cpp_a37a83060ac796961b44991c836f083f7}{B} with a supernodal LU factorization in the Pa\+StiX library. The matrix \doxylink{bench__gemm_8cpp_addc86e8508f14411ec98f521c520f875}{A} should be squared and nonsingular Pa\+StiX requires that the matrix \doxylink{bench__gemm_8cpp_addc86e8508f14411ec98f521c520f875}{A} has a symmetric structural pattern. This interface can symmetrize the input matrix otherwise. The vectors or matrices X and \doxylink{bench__gemm_8cpp_a37a83060ac796961b44991c836f083f7}{B} can be either dense or sparse.
