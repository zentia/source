\subsubsection*{template$<$typename \+\_\+\+Scalar$>$\newline
class Eigen\+::\+Diagonal\+Preconditioner$<$ \+\_\+\+Scalar $>$}
\doxylink{bench__gemm_8cpp_addc86e8508f14411ec98f521c520f875}{A} preconditioner based on the digonal entries 

This class allows to approximately solve for A.\+x = b problems assuming \doxylink{bench__gemm_8cpp_addc86e8508f14411ec98f521c520f875}{A} is a diagonal matrix. In other words, this preconditioner neglects all off diagonal entries and, in \doxylink{namespace_eigen}{Eigen}\textquotesingle{}s language, solves for\+: 
\begin{DoxyCode}{0}
\DoxyCodeLine{\mbox{\hyperlink{bench__gemm_8cpp_addc86e8508f14411ec98f521c520f875}{A}}.diagonal().asDiagonal()\ .\ \mbox{\hyperlink{offscreen_8c_a5fd331c99e778f04762be6d8173eb4d2}{x}}\ =\ \mbox{\hyperlink{offscreen_8c_a846c9667e34d56c560bb7f0ac6e173f6}{b}}}

\end{DoxyCode}



\begin{DoxyTemplParams}{模板参数}
{\em \+\_\+\+Scalar} & the type of the scalar.\\
\hline
\end{DoxyTemplParams}
\textbackslash{}implsparsesolverconcept

This preconditioner is suitable for both selfadjoint and general problems. The diagonal entries are pre-\/inverted and stored into a dense vector.

\begin{DoxyNote}{注解}
\doxylink{bench__gemm_8cpp_addc86e8508f14411ec98f521c520f875}{A} variant that has yet to be implemented would attempt to preserve the norm of each column.
