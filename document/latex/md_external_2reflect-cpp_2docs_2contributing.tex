\hypertarget{md_external_2reflect-cpp_2docs_2contributing_autotoc_md556}{}\doxysubsection{\texorpdfstring{Setup}{Setup}}\label{md_external_2reflect-cpp_2docs_2contributing_autotoc_md556}
Make sure includes are relative.

We need internal includes to be relative and not depend on any externally set include directory.

That is, for example, if you are within any file in {\ttfamily rfl/internal}, prefer 
\begin{DoxyCode}{0}
\DoxyCodeLine{\textcolor{preprocessor}{\#include\ "{}\mbox{\hyperlink{to__ptr__named__tuple_8hpp}{to\_ptr\_named\_tuple.hpp}}"{}}}

\end{DoxyCode}
 over 
\begin{DoxyCode}{0}
\DoxyCodeLine{\textcolor{preprocessor}{\#include\ "{}\mbox{\hyperlink{to__ptr__named__tuple_8hpp}{rfl/internal/to\_ptr\_named\_tuple.hpp}}"{}}}

\end{DoxyCode}
 For further details and reasoning, please refer to \href{https://github.com/getml/reflect-cpp/issues/30}{\texttt{ \#30}}.\hypertarget{md_external_2reflect-cpp_2docs_2contributing_autotoc_md557}{}\doxysubsection{\texorpdfstring{Compiling and running the tests}{Compiling and running the tests}}\label{md_external_2reflect-cpp_2docs_2contributing_autotoc_md557}
reflect-\/cpp uses vcpkg for dependency management, including gtest, which is required for the tests.


\begin{DoxyCode}{0}
\DoxyCodeLine{\#\ bootstrap\ vcpkg\ if\ you\ haven't\ done\ so\ already\ }
\DoxyCodeLine{git\ submodule\ update\ -\/-\/init}
\DoxyCodeLine{./vcpkg/bootstrap-\/vcpkg.sh\ \#\ Linux,\ macOS}
\DoxyCodeLine{./vcpkg/bootstrap-\/vcpkg.bat\ \#\ Windows}
\DoxyCodeLine{\#\ You\ may\ be\ prompted\ to\ install\ additional\ dependencies.}

\end{DoxyCode}
\hypertarget{md_external_2reflect-cpp_2docs_2contributing_autotoc_md558}{}\doxysubsection{\texorpdfstring{JSON only}{JSON only}}\label{md_external_2reflect-cpp_2docs_2contributing_autotoc_md558}
To compile the tests, do the following\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{cmake\ -\/S\ .\ -\/B\ build\ -\/DCMAKE\_BUILD\_TYPE=Release\ -\/DREFLECTCPP\_BUILD\_TESTS=ON}
\DoxyCodeLine{cmake\ -\/-\/build\ build\ -\/j\ 4\ \#\ gcc,\ clang}
\DoxyCodeLine{cmake\ -\/-\/build\ build\ -\/-\/config\ Release\ -\/j\ 4\ \#\ MSVC}

\end{DoxyCode}


To run the tests, do the following\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{./build/tests/json/reflect-\/cpp-\/json-\/tests}

\end{DoxyCode}
