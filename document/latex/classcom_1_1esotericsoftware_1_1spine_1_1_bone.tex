\doxysubsection{详细描述}
Stores a bone\textquotesingle{}s current pose. 

\doxylink{bench__gemm_8cpp_addc86e8508f14411ec98f521c520f875}{A} bone has a local transform which is used to compute its world transform. \doxylink{bench__gemm_8cpp_addc86e8508f14411ec98f521c520f875}{A} bone also has an applied transform, which is a local transform that can be applied to compute the world transform. The local transform and applied transform may differ if a constraint or application code modifies the world transform after it was computed from the local transform.
