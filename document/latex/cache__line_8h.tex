\doxysubsection{详细描述}
Cache-\/line size definition 

This file defines cache-\/line size constant. The constant is used many {\ttfamily libcds} algorithm to solve false-\/sharing problem.

The value of cache-\/line size must be power of two. You may define your cache-\/line size by defining macro -\/DCDS\+\_\+\+CACHE\+\_\+\+LINE\+\_\+\+SIZE=xx (where {\ttfamily xx} is 2\texorpdfstring{$\ast$}{*}\texorpdfstring{$\ast$}{*}N\+: 32, 64, 128,...) at compile-\/time or you may define your constant value just editing {\ttfamily \doxylink{cache__line_8h}{cds/user\+\_\+setup/cache\+\_\+line.\+h}} file.

The default value is 64.
